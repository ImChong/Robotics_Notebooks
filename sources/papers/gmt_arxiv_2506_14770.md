## 摘要级要点

- **问题：** 真机全身跟踪面临 **部分可观测**、**硬件不可行动作**、**MoCap 类别不平衡**、**MLP 表达力不足**。
- **框架：** 与 ExBody2 / OmniH2O 同族的 **两阶段教师–学生**：特权教师 PPO → 学生 DAgger（本体历史 + 运动目标）。
- **Adaptive Sampling：** 长动作随机裁剪（≤10 s，随机偏移 ≤2 s，周期性重裁）；按完成度 $c_i$ 与误差阈值 $E_i$ 定义采样级别，提高难样本有效采样率。
- **Motion MoE：** 软 MoE：门控对多专家动作分布加权求和；门控可视化显示复合技能上专家切换。
- **运动输入：** 局部（相对朝向）key body + 即时帧 ${\bm{g}}_t$ + 约 2 s 未来帧堆叠经 CNN 压成 ${\bm{z}}_t\in\mathbb{R}^{128}$（GMT-L2-M）。
- **数据策展：** 规则过滤不可行动作 → 初步策略完成率再过滤 → 8925 clips / 33.12 h。
- **仿真结果（Table 2，特权策略）：** AMASS-Test / LAFAN1 上相对重训 ExBody2 在 $E_{\mathrm{mpkpe}}$、$E_{\mathrm{mpjpe}}$、$E_{\mathrm{vel}}$、$E_{\mathrm{yaw\,vel}}$ 全面更优；消融显示去掉 MoE 或 Adaptive Sampling 均变差。
- **真机：** 风格化走、高踢、舞、转、蹲走、射门等（见项目页）。
- **应用：** MuJoCo 上可跟踪 **MDM 文本生成动作**。
- **局限：** 缺接触丰富技能（起身/翻滚）；无地形观测，不适斜坡/楼梯。

### 2) Adaptive Sampling（要点）

- Random Clipping：>10 s 切分；随机偏移 ≤2 s；训练中周期性重裁。
- $c_i$：完成度，初始 10，每次完成 ×0.99，下限 1；$E_i$ 随 $c_i$ 从 0.25 退火到更严。
- 采样概率由 $s_i$ 归一化得到，从训练初期即偏向难片段。

### 3) Motion MoE

- 专家与门控同吃 **本体 + 运动目标**；${\bm{a}}=\sum_i p_i {\bm{a}}_i$。
- 消融与 top-percentile 误差：MoE 对 **更难动作** 收益更大。

### 4) 观测与训练细节（附录）

| 项 | 内容 |
|----|------|
| 教师观测 | 本体 ${\bm{o}}_t$ + 特权 ${\bm{e}}_t$ + 运动目标 |
| 学生观测 | ${\bm{o}}_t$、历史 ${\bm{o}}_{t-20:t}$、运动目标 |
| 动作 | 目标关节位置 → PD |
| 仿真 | IsaacGym，4096 envs；控制 50 Hz / 仿真 500 Hz |
| Sim2Real | DR + action delay；减速器反射惯量 `armature = k²I` |
| 算力（文中） | 教师 ~3 天 / RTX 4090；学生 ~1 天 |
