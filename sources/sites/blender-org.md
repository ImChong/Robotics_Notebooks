- **抓取说明**：以 **2026-06-07** 对首页公开文案与导航结构的抓取为准；版本号与新闻条目会随发布周期更新。

## 一句话

**Blender** 是免费开源的全流程 **3D 创作套件**（建模、雕刻、UV、绑定、动画、模拟、渲染、合成、运动跟踪与视频编辑），由贡献者社区在 **GNU GPL** 下维护，并通过 **Blender Studio** 开放电影与 **Blender Conference** 等生态活动持续迭代。

## 为什么值得保留

- 机器人知识库中多处出现 **Blender** 作为 **DCC（数字内容创作）**、**BVH/网格导出**、**USD 场景组装** 与 **插件宿主**（如 [ppf-contact-solver](../repos/ppf-contact-solver.md)、[SAM3DBody-cpp](../repos/sam3dbody-cpp.md)），但此前缺少独立溯源页。
- 与 [Mixamo](./mixamo.md)（商业在线角色库）、[NVIDIA Omniverse](../wiki/entities/nvidia-omniverse.md)（USD 协作仿真底座）形成 **资产创作 → 格式交换 → 仿真/训练** 链路上的对照节点。
- 官网明确产品能力分区：**Cycles** 渲染、**Geometry Nodes** 程序化建模、**VFX 跟踪**、**绑定与动画**、**Grease Pencil** 2D/3D 混合、**Python 可扩展界面**——与具身智能中的场景生成、动捕后处理、可视化管线直接相关。

## 公开功能要点（来自官网首页，2026-06-07）

| 模块 | 代表能力 |
|------|----------|
| **建模 / 雕刻 / UV** | N-Gon、动态细分、多分辨率雕刻、纹理绘制与遮罩 |
| **动画与绑定** | 包络/骨架/自动蒙皮、B-spline 骨骼、曲线编辑器、dope sheet、音画同步 |
| **VFX / 跟踪** | 相机与物体跟踪、平面/三脚架求解器、实拍与 3D 场景实时预览 |
| **渲染** | 内置 **Cycles** 路径追踪；CPU/GPU、PBR/HDR、VR 渲染；视口实时预览 |
| **2D / Story Art** | Grease Pencil：视口内 2D 绘制与 3D 场景结合、洋葱皮动画 |
| **扩展** | Python 控制界面布局与工具；社区 **add-on** 生态 |
| **组织与生态** | ASWF、Khronos、Linux Foundation、OIN 成员；AMD/Apple/Intel/NVIDIA/Qualcomm 等硬件厂商支持 |

## 近期开发动态（官网新闻摘要）

- **Blender 5.2 LTS**：多图像纹理场景渲染的 **内存效率** 改进（开发中）。
- **Winter of Quality 2026**（2025–2026 北半球冬季）：团队优先 **稳定性与整体质量**。
- **Geometry Nodes Workshop**（2025-09）：程序化节点工作流专题讨论。
- **Google Summer of Code 2025**：8 个入选项目结果公布。
- **Blender Studio**：开放电影 **Singularity**（4K HDR）发布与首映活动。
