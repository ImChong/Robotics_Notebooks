论文导航

这页不是论文仓库目录，而是一个真正能顺着读下去的导航页：按问题分类、按主线组织、按进入顺序给入口，尽量避免“收藏几百篇，结果一篇都没读透”。

怎么使用这页

最稳的方式不是到处跳，而是先沿一条主线打穿，再横向扩展。

如果你当前主攻 humanoid locomotion

先读 RL → Sim2Real → Humanoid，最后再回到 IL 看参考动作与技能迁移。

如果你想补技能学习方法论

先读 IL 和 RL 的对照，再看 Humanoid 里混合范式的代表工作。

如果你想补控制主线

先从 Motion Control 打骨架，再看 RL / IL 如何建立在控制主线之上。

如果你怕读散

每个分类先抓 2–4 篇代表作，别上来就全刷。读完一组，回模块页做结构化总结更值钱。

RL：强化学习主线

适合想把 locomotion、鲁棒性训练、长期回报优化打透的人。重点不是算法百科，而是机器人控制任务里 RL 为什么有效、怎么稳定、怎么部署。

进入 RL 模块页

建议先看

先在模块页里建立 state / action / reward / sim2real 的整体框架，再去读论文，不然很容易只记住算法名。

PPO / Locomotion 代表作

第一批必读

重点看大规模并行仿真、策略结构、奖励分解和训练稳定性，理解为什么 PPO 成了很多 humanoid locomotion 的工程默认选项。

Robust Locomotion / Disturbance Recovery

第二批建议读

重点看抗扰、恢复、复杂地形、速度变化和接触切换，不要只盯“在平地上能走”。

RL + Motion Prior

进阶

当你开始关心“既能走，又要自然、协调、像人”，就要看 RL 和 imitation / motion prior 的混合路线。

IL：模仿学习主线

适合想利用动捕、遥操作、专家轨迹、已有策略数据来加速技能学习的人。核心问题是：怎么把演示变成可泛化、可部署的控制能力。

进入 IL 模块页

建议先看

先建立对 behavior cloning、distribution shift、retargeting、tracking pipeline 的理解，再决定往哪类论文深挖。

Behavior Cloning / DAgger

第一批必读

先把最基础的 supervised imitation 和 online aggregation 逻辑吃透，不然后面看 fancy 模型会飘。

Motion Retarget / Reference Tracking

第二批建议读

重点看人类动作到机器人动作的转换、参考动作跟踪和闭环稳定性，这部分跟 humanoid 特别相关。

Diffusion Policy / Sequence Policy

进阶

适合关心复杂时序行为、多峰动作分布和视觉条件控制的场景，尤其在 manipulation 里很值钱。

Motion Control：运动控制主线

如果骨架没搭起来，后面很多 RL / IL 论文会看得很虚。控制主线是整个理解体系的底板。

进入 Motion Control 模块页

建议先看

从质心动力学、任务空间控制、全身控制、步态与平衡开始，把控制问题的语言体系搭起来。

WBC / QP / TSID

第一批必读

理解任务优先级、约束、接触力分配、可行性边界，这些是机器人控制论文里的基本语法。

MPC / Optimal Control

第二批建议读

重点看如何把系统动力学、目标函数和约束写成可以在线求解的问题。

Learning-augmented Control

进阶

当你开始关注学习控制与经典控制的结合，这条线会非常重要。

Sim2Real：从仿真到真机

这是最容易被一句“成功迁移到真机”糊弄过去的部分，但它恰恰最值钱。

Domain Randomization

第一批必读

看摩擦、质量、延迟、噪声、地形扰动这些随机化是怎么帮助策略获得鲁棒性的。

Actuator Modeling / Delay Modeling

第二批建议读

很多真机成败就卡在执行器和延迟上。没把这部分看明白，部署很容易扑街。

System Identification

补充线

不一定追求“唯一最准参数”，但理解系统辨识怎么辅助部署很有必要。

Deployment Reports / Survey

强烈建议读

多看真实部署经验总结。很多最关键的坑，往往写在 appendix、supplementary 或工程报告里。

Humanoid：人形机器人专项

这里不是再重复 RL / IL / Control，而是看它们在人形系统里如何真正缠在一起工作。

Locomotion

先读

站、走、跑、转向、抗扰、复杂地形，先打透最核心主战场。

Motion Imitation / Style Control

再读

重点看 reference motion、style prior、AMP、tracking reward，理解“像人一样动”怎么来的。

Whole-body Skills

进阶

上肢参与、全身协调、多接触交互，开始逼近更真实的人形能力。

Deployment / Hardware Reports

别跳过

真正决定你有没有工程直觉的，往往不是 headline，而是部署细节和失败经验。

开源生态与工具链入口

论文不要孤立看，最好和代码仓、仿真平台、训练框架一起看，这样理解会快很多。

Simulation

MuJoCo、Isaac Gym、Isaac Lab、Isaac Sim、Gazebo / GZ。重点不是都学，而是知道各自适合什么工作流。

RL Frameworks

legged_gym、IsaacGymEnvs、humanoid-gym 等，是理解实际训练工程的好入口。

Humanoid Projects

看真实项目如何组织 observation、action、reward、deploy pipeline，比只看算法摘要值钱得多。

Benchmark / Survey

用 benchmark 看共性，用 survey 建地图，但别停留在 survey 层。最终还是得回到代表工作和代码。
