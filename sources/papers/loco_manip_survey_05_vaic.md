# VAIC: Vision-Guided Humanoid Agile Object Interaction Control via Decoupled Commands

> 来源归档（ingest · Loco-Manip 8 篇周报 第 05/8）

- **标题：** VAIC: Vision-Guided Humanoid Agile Object Interaction Control via Decoupled Commands
- **类型：** paper
- **Loco-Manip 分类：** 03 命令空间与控制器
- **机构：** 清华大学；香港科技大学（广州）；小米机器人实验室
