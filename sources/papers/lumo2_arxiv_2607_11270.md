# Lumo-2：Towards Predictive, Aligned, and Scalable Robot Learning（技术报告）

> 来源归档（ingest）

- **标题：** Towards Predictive, Aligned, and Scalable Robot Learning（Lumo-2）
- **类型：** paper（技术报告）
- **arXiv abs：** <https://arxiv.org/abs/2607.11270>
- **项目页：** <https://www.astribot.com/research/Lumo2>
- **机构：** Astribot Team（星尘智能）
- **骨干：** **Qwen3.5-4B** 持续训练；统一多模态 Transformer 联合建模 **潜空间世界动力学 φ**、动作 chunk 与 VQA 文本
- **入库日期：** 2026-07-16
- **一句话说明：** **潜空间世界–动作模型**：以物理接地 **latent world dynamics** 作隐式推理桥，**三阶段动作 token 预对齐**（动力学→视觉语言→VLWA 共训）；在 **Astribot S1** 上 **22 项** 挑战真机任务全面超 **π₀.₅** 与 **Fast-WAM**；**BAR** 解码 **2.71×** 加速；支持 VisionPro / egocentric 人视频 **无专用迁移机制** 的 emergent 人–机共训增益。

## 核心摘录（面向 wiki 编译）

### 1) 相对 Lumo-1 的范式迁移

- **链接：** §1 Introduction；Related Work
- **摘录要点：** Lumo-1 用 **显式结构化文本推理**（粗到细规划）引导动作，灵活性与可扩展性受限、推理延迟高；Lumo-2 以 **latent world dynamics** 作隐式推理，token 复杂度更低、容量更高，且自然形成 **跨本体语义一致** 的 action-inducible 未来空间。
- **对 wiki 的映射：**

  - [World Action Models](../../wiki/concepts/world-action-models.md) — latent Joint WAM 坐标

### 2) 三阶段训练配方

- **链接：** §3 Training Recipe
- **摘录要点：**
  - **Stage 1（30k steps, 64×H100）：** 双分支 VQ——DINOv2 冻结提特征 + 因果时空 Transformer 学 **φ**；动作按躯干/双臂分组再 VQ；**φ→A 双向约束** 动作重建；混合 **Astribot S1 + 跨本体机器人 + Ego4D/EPIC/EgoTaskQA** 等。
  - **Stage 2（12k steps）：** 冻结 Stage1 编码器，新增 **语义动作码本** + projector；多任务：动作重建、行为理解（V+A→语言）、V+L 引导动作、跨模态预测、对比/判别对齐。
  - **Stage 3（120k steps, 160×H100）：** **~53M** VLM 样本 + 视频动力学 + 机器人 **VLA/VLW/VLWA** 混训；历史动作队列 **context differentiation**；**BAR** 块级并行解码。
- **对 wiki 的映射：**
  - [Lumo-2](../../wiki/entities/lumo-2.md) — 三阶段 Mermaid 与数据混合

### 3) 22 项真机评测任务（Tab. 4–5）

- **链接：** §4.3 Task Post-Training
- **摘录要点：** 三大能力维：**时序推理**（动态场景反应 / 运动预测 / 记忆）、**物理理解**、**控制复杂度**（长程 / 灵巧）。任务含：配对笔帽、拆包、装书包（衣/玩具）、提垃圾袋、装箱、熨烫挂衣、做咖啡/鸡尾酒、传送带捡蛋、旋转架挂杯/放方块、钓鱼、翻蛋、砸钉、擦白板、倒水、备蛋等；每任务 **10 布局 ×2 次** 盲评，**归一化子任务得分**。
- **对 wiki 的映射：**

### 4) 关键定量结果

- **链接：** §4.1–4.5
- **摘录要点：**
  - **具身 VLM：** Lumo-2 VLM 相对 Qwen3.5-4B 在 VSIBench、MindCube、RefSpatial 等大幅上升；Stage3 共训未灾难性遗忘纯 VLM。
  - **Pick-and-place：** Basic / Unseen Instructions / Unseen Objects 三设定均超 **π₀.₅**、**Fast-WAM**；未见指令/物体增益最大。
  - **VLWA vs VLA 微调：** 传送带捡蛋 **100% vs 92%**；旋转架放方块 **81.67% vs 74.17%**。
  - **人–机迁移：** VisionPro 单视角有动作重定向 + 多视角无标注 egocentric 视频，共训后在 **unseen objects** pick-place 显著增益。
  - **BAR 延迟：** 端到端 **93.53 ms**（AR **253.66 ms**）。
- **对 wiki 的映射：**
  - [Lumo-2](../../wiki/entities/lumo-2.md) — 结果摘要表
  - [VLA](../../wiki/methods/vla.md) — π₀.₅ / Fast-WAM 对照索引

### 5) 动作 token 几何与 scaling

- **链接：** §4.5
- **摘录要点：** 纯重建 **Action Only** tokenizer 在下游控制上与重建精度 **misaligned**；Stage2 语义对齐后 **SO(3)/XYZ 重建误差更低**、原子动作线性探测 **84.1%→95.0%**；Stage2 初始化在 General PnP 上 **更快更低解码误差**。
