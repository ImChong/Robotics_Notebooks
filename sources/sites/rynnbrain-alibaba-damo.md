- **代码：** **已开源** — GitHub `alibaba-damo-academy/RynnBrain`（Apache-2.0）；含 transformers / SGLang 推理、Gradio `demo.py`、`cookbooks/` notebook。
- **权重：** **已发布** — RynnBrain1.1-2B / 9B / 122B-A10B（HF + ModelScope）。
