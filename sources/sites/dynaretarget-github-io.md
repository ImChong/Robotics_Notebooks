## 一句话摘要

Atari Lab 等人提出的 **DynaRetarget** 官方站点：展示 **IK → SBTO 动态精炼 → RL 跟踪 → 真机零样本** 管线在踢箱、搬运、推物、手递等 **contact-rich loco-manipulation** 上的视频与方法概览；链出 arXiv 与 **SBTO 开源实现**。
