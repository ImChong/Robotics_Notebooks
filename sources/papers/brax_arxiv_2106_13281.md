# Brax — A Differentiable Physics Engine for Large Scale Rigid Body Simulation（arXiv:2106.13281）

> 来源归档

- **标题：** Brax — A Differentiable Physics Engine for Large Scale Rigid Body Simulation
- **类型：** paper
- **作者：** C. Daniel Freeman, Erik Frey, Anton Raichuk, Sertan Girgin, Igor Mordatch, Olivier Bachem
- **链接：** https://arxiv.org/abs/2106.13281
- **会议：** NeurIPS 2021 Datasets and Benchmarks Track
- **入库日期：** 2026-05-18
- **一句话说明：** 提出在加速器上高吞吐、端到端可微的刚体仿真与 RL 训练一体化设计，并与后续 **MJX 承接 MuJoCo 级物理** 的路线形成对照阅读材料。
- **沉淀到 wiki：** 是 → [`wiki/entities/brax.md`](../../wiki/entities/brax.md)

## 核心主张（归纳）

- 目标是把 **大规模并行仿真** 与 **基于自动微分的算法**（如解析策略梯度）放在 **同一 JAX 程序** 中，降低 Python 循环与 CPU–加速器拷贝带来的墙钟瓶颈。
- 通过 **多套可替换物理管线**（论文与 README 中广义坐标、PBD、弹簧等路线）共享环境 API，便于做 **不同物理近似** 下的迁移与对比实验。
