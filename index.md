# Robotics Notebooks Index

这是 `Robotics_Notebooks` 的知识索引首页。

当前项目从“资源型思维导图”重构为一个**以运动控制为切入口、通向机器人全栈工程能力**的技术栈导航项目。

## 项目定位

- **终极目标：** 全栈机器人工程师
- **当前切入口：** 机器人运动控制算法工程师
- **当前重点：** 运动控制、强化学习、模仿学习、人形机器人

## v1 结构入口

- [项目目录结构 v1](STRUCTURE_v1.md)
- [README](README.md)

## 项目结构

- `wiki/`：结构化知识页
- `sources/`：原始资料
- `roadmap/`：成长路线与阶段导航
- `tech-map/`：技术栈地图与依赖关系
- `references/`：论文导航与开源生态
- `docs/homepage-copy-v1.md`：首页文案草稿
- `schema/`：知识库规则
- `exports/`：面向网页渲染的导出层
- `log.md`：重构与更新日志

## 当前优先方向

第一阶段重点不是把网页做花，而是把知识骨架、模块依赖、学习路径搭清楚。

### 内圈主攻
- Humanoid Control
- Reinforcement Learning
- Imitation Learning
- Sim2Real
- Locomotion
- Whole-Body Control

### 外圈扩展
- Perception
- Planning
- ROS2 / Middleware
- Deployment
- System Integration
