## 一句话

**SMPLOlympics** 官方项目页：展示仿真人形在多项奥运风格运动上的 qualitative 结果、环境与基线说明，并指向 GitHub 代码与 arXiv 论文。

## 为什么值得保留

- **一手入口**：与论文 PDF 并列，便于核对各运动任务的演示视频与补充材料索引。
- **与 SMPL 生态对齐**：强调人形与 **SMPL/SMPL-X** 兼容，便于对接 TRAM、PHC、AMASS 等社区数据管线。
