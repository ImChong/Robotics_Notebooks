# WOLF-VLA: Whole-Body Humanoid Optimal Locomotion Framework for Vision-Language-Action Learning

> 来源归档（ingest）

- **标题：** WOLF-VLA: Whole-Body Humanoid Optimal Locomotion Framework for Vision-Language-Action Learning
- **类型：** paper
- **原始链接：** <https://arxiv.org/abs/2606.25591>
- **机构：** DFKI；University of Oldenburg 等
- **入库日期：** 2026-07-03
- **一句话说明：** 用 **全身最优控制（OCP）** 合成六类 locomotion 相关任务的 **动态一致** 轨迹数据集，训练 **VLA** 从自然语言与 ego 视觉直接生成人形运动策略；强调数据 **最优性、安全性与接触一致性**。
