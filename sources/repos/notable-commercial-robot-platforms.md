| 组织 | 代表产品 / 方向 | 官方入口（2026-05 核对） |

| Boston Dynamics | Atlas（液压退役后主推全电 Atlas）、Stretch | https://www.bostondynamics.com/ |
| Agility Robotics | Digit（物流人形） | https://www.agilityrobotics.com/ |
| Figure AI | Figure 02、Helix | https://www.figure.ai/ |
| 1X Technologies | EVE（轮式人形）、NEO（双足家庭向） | https://www.1x.tech/ |
| Tesla | Optimus | https://www.tesla.com/AI |

| Sanctuary AI | Phoenix、Carbon AI | https://www.sanctuary.ai/ |
