# World-Ego Modeling for Long-Horizon Evolution in Hybrid Embodied Tasks（arXiv:2605.19957）

> 来源归档（ingest）

- **标题：** World-Ego Modeling for Long-Horizon Evolution in Hybrid Embodied Tasks
- **缩写：** **WEM**（World-Ego Model）；范式名 **World-Ego Modeling**
- **类型：** paper / 具身世界模型 / 视频生成 / 长程混合导航–操作
- **arXiv：** <https://arxiv.org/abs/2605.19957>（PDF：<https://arxiv.org/pdf/2605.19957>）
- **项目页：** <https://zgca-hmi-lab.github.io/WEM/>
- **代码：** <https://github.com/ZGCA-HMI-Lab/WEM>
- **作者：** Zuyao Lin, Jianhui Zhang, Peidong Jia, Xiaoguang Zhao, Shanghang Zhang, Xingyu Chen（中科院自动化所、国科大、中关村学院、上海交大、北大等；✉ 通讯作者以论文为准）
- **入库日期：** 2026-05-22
- **一句话说明：** 提出 **World-Ego Modeling** 范式，将具身未来演化解耦为 **指令无关的场景世界（world）** 与 **指令条件的机体自我（ego）**；实例化 **WEM**（RCA 隐式 world–ego 规划器 + CP-MoE 级联并行扩散生成器），并发布基于 **BEHAVIOR-1K** 的 **HTEWorld** 长程导航–操作混合基准（125K 训练片段、300 条多轮评测轨迹）。

## 摘要级要点

- **问题：** 现有视频世界模型常在 **单一流** 中同时预测「世界」与「自我」，导致 **world–ego 纠缠**；在长程、**导航与操作交错** 的混合具身任务中，场景一致性（导航侧）与接触丰富动力学（操作侧）互相干扰，性能退化。
- **主张：** **World** 承载持久、**指令无关** 的场景规律（布局、物体恒常性等）；**Ego** 承载 **机器人中心、指令条件** 的动力学（本体运动、物体交互）。二者应在建模与生成结构上 **显式解耦**。
- **World–Ego 边界（三视角）：**
  1. **Motion-based：** 用光流等运动线索，将接触引起的物体运动归 ego，其余归 world。
  2. **Semantic-based（WEM 默认）：** 学习语义掩码，场景区域归 world，机器人/被操作物体归 ego；默认用 **实例分割** 构造掩码。
  3. **Intention-based：** world 侧用视觉历史规律，ego 侧用指令条件动力学，由生成器隐式学习各区域如何消费两类信息（论文中分离较弱）。
- **解耦策略（三档）：** **Post-** / **Pre-** / **Full disentanglement**；**Full** 在路由、专家分工与 unrouting 上结构分离最强，配合 **semantic view** 为 WEM 默认。
- **WEM 架构：**
  - **预测阶段：** 预训练 **VLM** 作状态预测器，经 **Role-Conditioned Attention（RCA）** 与 **非对称 query 预算** 从视觉–语言 token 推断分离的 world / ego 状态；**Semantic Head** 预测 world–ego 代理掩码。
  - **生成阶段：** **Cascade-Parallel Mixture-of-Experts（CP-MoE）** 扩散生成器：前级专家预测 world–ego 代理，后级按 **pre / post / full** 策略分流；**Neighbor-expanded routing** 扩展语义边界邻域；**chunk-wise 自回归扩散** 产出长程视频（评测协议：每 chunk **37** 帧 @ **480×480**、**16 FPS**、**35** 扩散步等，见附录）。
- **HTEWorld：** 基于 **BEHAVIOR-1K** 构建的首个 **长程混合导航–操作** 世界建模基准；**125K** 训练视频片段（**450 万+** 帧）、细粒度动作标注；**300** 条多轮评测轨迹（**2000+** 指令）。指标含 **WorldArena** 归一化套件 **EWMScore** 与导航–操作专用六项（chunk 边界动力学、晚前缀状态对齐、指令步检索等）。
- **主要结果（HTEWorld / WorldArena）：** WEM **EWMScore 61.48**，优于同数据微调的 **WoW-7B（53.44）**、**Cosmos-Predict 2.5-14B（55.41）**、**PAN-style（58.40）** 等；在既有 **操纵导向** 基准上仍具竞争力。消融：去掉非对称 query、RCA 或邻域扩展路由均显著降分。
- **局限（附录）：** 主要在 **仿真** 数据上评测；边界依赖 **语义掩码预处理**；极长自回归仍累积误差；与 JEPA / GEM / Cosmos / WoW 等相邻工作对照见 Related Work。

## 核心论文摘录（MVP）

### 1) World–Ego 纠缠与混合长程任务动机

- **链接：** <https://arxiv.org/abs/2605.19957> §1 Introduction
- **摘录要点：** 世界模型需同时预测环境与机体，但二者对应 **意图无关变化 vs 意图驱动行为**；混合导航–manipulation 中，导航依赖场景持久性，操作依赖接触动力学，单流建模易在长程多轮指令下失稳。

### 2) 三边界 × 三解耦 × WEM 实例化

- **链接：** 项目页 Framework / Model 图；论文 §3 World-Ego Modeling
- **摘录要点：** 通用框架 = **VLM 状态预测器** + **CP-MoE 生成器**；默认 **semantic + full disentanglement**；RCA 用角色条件注意力与不对称 query 组分离 world/ego 状态。

- **链接：** 项目页 Quantitative Results；论文 §4 实验
