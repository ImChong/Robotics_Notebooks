## 一句话摘要

Sber Robotics Center **Green-VLA** 官方站点：展示 **Green 人形** 语言条件双手分拣/清理、**电商货架 JPM 引导**、**五阶段训练** 消融与 **Simpler / CALVIN** 基准对比视频。

## 公开信息要点（截至入库日）

- **核心叙事：** 在 **~3000 h** 统一机器人数据（少于 π₀ 的 >10k h）上，通过 **DataQA + 语义统一动作 + 分阶段课程 + R2 RL** 达到或超过多项 foundation policy。
- **演示板块（归纳）：** 人形 OOD 布局操作、任务规划分解子任务、电商 SKU 级拣选、双臂 Cobot 清理桌、WidowX R0→R1→R2 递进。
- **资源链接：** arXiv、GitHub、PDF；贡献者按 VLA / RL / Data / Control / Collection 分组致谢。
