# Page Types

本文件定义 `wiki/` 中的页面类型。

## 1. Overview Page
回答：一个领域整体是什么。

适用场景：
- robot learning overview
- humanoid control overview
- sim2real overview

推荐结构：
1. 一句话总结
2. 这个领域解决什么问题
3. 核心组成
4. 和其他领域的关系
5. 推荐起步阅读

## 2. Concept Page
回答：一个概念是什么。

适用场景：
- Sim2Real
- Whole-Body Control
- MDP
- Domain Randomization

推荐结构：
1. 一句话定义
2. 为什么重要
3. 核心组成
4. 常见误区
5. 关联页面
6. 推荐继续阅读

## 3. Method Page
回答：一种方法怎么工作。

适用场景：
- Reinforcement Learning
- Imitation Learning
- MPC
- TSID

推荐结构：
1. 解决什么问题
2. 输入/输出
3. 核心思想
4. 优势
5. 局限
6. 在机器人中的典型应用
7. 关联页面

## 4. Task Page
回答：一个任务方向需要什么能力。

适用场景：
- locomotion
- manipulation
- loco-manipulation
- teleoperation

推荐结构：
1. 任务定义
2. 关键挑战
3. 常见方法路线
4. 评价指标
5. 关联系统/方法

## 5. Comparison Page
回答：两个或多个方法/框架如何选型。

适用场景：
- WBC vs RL
- PPO vs SAC
- MPC vs policy learning

推荐结构：
1. 比较对象
2. 核心差异
3. 各自适用场景
4. 常见误判
5. 结论

## 6. Roadmap Page
回答：学习或研究该主题的顺序。

适用场景：
- humanoid control roadmap
- sim2real roadmap
- control engineer to robot learning roadmap

推荐结构：
1. 适合谁
2. 先修知识
3. 学习阶段划分
4. 每阶段推荐资源
5. 常见卡点

## 7. Source Summary Page
回答：某篇论文/博客/课程讲了什么。

注意：
- 这里只做简版摘要与定位说明
- 深度论文拆解尽量放在 [`Humanoid_Robot_Learning_Paper_Notebooks`](https://github.com/ImChong/Humanoid_Robot_Learning_Paper_Notebooks)
