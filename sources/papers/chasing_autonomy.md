## 核心论文摘录（MVP）

### 1) Chasing Autonomy: Dynamic Retargeting and Control Guided RL for Performant and Controllable Humanoid Running (Olkin et al., 2026)
- **链接：** <https://arxiv.org/abs/2603.25902>
- **核心贡献：** Presents a pipeline to dynamically retarget human motions through optimization with hard constraints, generating improved periodic reference libraries from single human demonstrations. Studies the effect of reference motion and reward structure, concluding that goal-conditioned and control-guided reward tracking dynamically optimized human data performs best. Demonstrated up to 3.3 m/s running speed on Unitree G1 robot outdoors with obstacle avoidance.
