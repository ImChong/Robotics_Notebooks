## 项目页公开主张（归纳）

- **能力：** Generation 与 Retrieval；模态切换（Point Cloud / PC+Normal / Mesh）；复杂度档位可视化；可切换显示 **CAD program** 文本。
- **SynthBal：** 页面展示相对原始不平衡数据集的生成质量对比叙事。
- **作者机构：** MIT（Yu、Alam、Hart、Ahmed）。
