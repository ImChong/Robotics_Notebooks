# Sources

这里是 `Robotics_Notebooks` 的原始资料层。

目标不是直接回答问题，而是作为知识库的输入来源。

## 目录说明

- `papers/`：论文来源
- `blogs/`：博客/文章
- `courses/`：课程与公开视频
- `videos/`：视频资源
- `repos/`：代码仓库
- `notes/`：历史笔记、旧版资源整理、临时摘录

## 当前入口

### 旧版资源归档
- [legacy-readme-resource-map.md](notes/legacy-readme-resource-map.md)

这个文件保留了旧 README 中原有的大量资源链接，避免重构过程中信息丢失。

## 使用原则

1. 新资料优先先进入 `sources/`
2. 真正沉淀后的知识，再进入 `wiki/`
3. 不把 `wiki/` 写成纯链接堆
4. 不再让根目录 `README.md` 承担所有资源导航职责

## 下一步建议

后续可以继续把 `legacy-readme-resource-map.md` 中的内容按主题拆分到：
- `papers/`
- `courses/`

第一阶段先完成“把大 README 瘦身”，第二阶段再做细分整理。
