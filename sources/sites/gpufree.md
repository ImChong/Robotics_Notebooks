# 算力自由（GPUFree）

> 来源归档

- **标题：** 算力自由 GPU 云算力平台
- **类型：** site（国内 GPU 云算力平台）
- **来源：** 北京算力自由科技有限公司（2024 年成立）
- **链接：** https://www.gpufree.cn/home 、https://www.gpufree.cn/docs/
- **入库日期：** 2026-07-02
- **一句话说明：** 国内 GPU 算力调度云平台，整合多 IDC 资源提供容器化开发机；支持按需（按秒）与包时长租用，强调 L40/L40S 等大显存企业卡与具身仿真友好镜像（含 noVNC-Vulkan 桌面）。

## 为什么值得保留

- **机器人仿真显式指引**：官方快速开始写明 **具身智能机器人仿真不宜选无 RT 核心的 A100/H100**（无法支撑显示与实时渲染），并推荐 **noVNC-vulkan** 等带桌面镜像——与 [Isaac Sim](../../wiki/entities/isaac-lab.md) / 图形仿真训练直接相关。
- **大显存企业卡供给**：首页主推 **L40-48G / L40S-48G**（48GB 显存），适合大 batch Isaac Lab、VLA 微调等 **显存敏感**任务。
- **计费与存储模型清晰**：计算资源仅 **运行中** 计费；关机仍收存储费；按需实例关机 **15 天**后自动释放。

## 平台要点（文档 2026-07 摘录）

### 公司定位

- 使命：「让算力像电一样便捷、自由地使用」。
- 自研 GPU 调度云，整合多地 IDC / 运营商资源；规划「算力 & 算法交易市场」（算力租赁 + 模型交易 + 算法共享）。

### 开发机实例

- **容器虚拟化**：物理多卡服务器切分为 1–N 卡远程开发机。
- **资源配比**：以 GPU 为中心分配 CPU / 内存（例：4090×8 主机每卡 12 核 CPU、50GB 内存）。
- **内置应用**：SSH（`sshd`）、Jupyter；镜像可带 ComfyUI、机器人仿真 VNC 等。

### 存储布局

| 类型 | 免费容量 | 挂载位置 | 说明 |
|------|----------|----------|------|
| 系统盘 | 30GB | `/` | 关机保留；释放实例时删除 |
| 数据盘 | 100GB | `/root/gpufree-data` | 可扩容 |
| 公共存储 | 因机房而异 | `/root/gpufree-data/share` | 跨机共享，内网较慢 |

扩展存储：**0.01 元/GB/自然日**（免费额度外）；关机仍计费。

### 计费模式

- **按需使用**：后付费，**仅运行中**扣计算费（文档称按秒粒度）。
- **包时长**：预付费预留算力卡时段。
- **存储**：与开关机无关；不用须 **释放实例** 以免持续扣费。

### 创建实例两条路径

1. **从显卡型号**：算力市场 → 选卡型、区域、卡数、数据盘、CUDA 对应基础镜像或桌面镜像。
2. **从镜像**：镜像市场 → 收藏常用框架 / 仿真镜像 → 快速创建（部分镜像需拉取等待）。

### 机器人相关提示（官方文档）

- 仿真需 **RT 核心**支持显示：避免纯计算卡做图形仿真。
- 可选 **开发 tag** 按 CUDA 版本选基础镜像，或 **noVNC-vulkan** 远程桌面跑仿真 GUI。
