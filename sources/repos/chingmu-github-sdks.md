# ChingMuVisionTech（GitHub SDK 组织）

> 来源归档

- **标题：** CHINGMU | MOCAP — GitHub 集成 SDK
- **类型：** repo（组织）
- **链接：** https://github.com/ChingMuVisionTech
- **关联官网：** https://en.chingmu.com/
- **入库日期：** 2026-07-20
- **一句话说明：** 青瞳视觉在 GitHub 发布的 **引擎/语言集成 SDK 与插件**（非核心 CMAvatar 源码）；覆盖 UE、Unity、C++ VRPN、Python、iClone、MotionBuilder 等，用于把光学动捕流接入实时渲染与第三方管线。

## 主要公开仓库（截至 2026-07-20）

| 仓库 | 用途 |
|------|------|
| [ChingMuUeSDKs](https://github.com/ChingMuVisionTech/ChingMuUeSDKs) | Unreal Engine 5.x 插件（CMMotionCapture） |
| [ChingMuUnitySDKs](https://github.com/ChingMuVisionTech/ChingMuUnitySDKs) | Unity 2021+ SDK |
| [ChingMuCppSDKs](https://github.com/ChingMuVisionTech/ChingMuCppSDKs) | C++ VRPN 数据获取示例 |
| [ChingMuPythonSDKs](https://github.com/ChingMuVisionTech/ChingMuPythonSDKs) | Python 集成 |
| [ChingMuiCloneSDKs](https://github.com/ChingMuVisionTech/ChingMuiCloneSDKs) | iClone 8 插件 |
| [MotionBuilder-Plugins](https://github.com/ChingMuVisionTech/MotionBuilder-Plugins) | MotionBuilder 插件 |
| [Chingmu-Raspberry-Pi](https://github.com/ChingMuVisionTech/Chingmu-Raspberry-Pi) | 树莓派相关示例 |

## 开放边界

- **已发布：** 各引擎插件包、帮助文档与教程视频（多为 Release 附件）。
- **未发布：** 光学求解器、CMAvatar 核心处理与相机固件；机器人侧 **ROS/ROS2 桥接** 在官网叙述为集成能力，但未见对应开源仓库。
