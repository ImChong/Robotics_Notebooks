# HDMI 项目页

> 来源归档（site / project page）

- **标题：** HDMI: Learning Interactive Humanoid Whole-Body Control from Human Videos
- **类型：** project page
- **URL：** <https://hdmi-humanoid.github.io>
- **论文：** <https://arxiv.org/abs/2509.16757>
- **代码：** <https://github.com/LeCAR-Lab/HDMI>
- **视频：** <https://www.youtube.com/watch?v=GvIBzM7ieaA>；35 min door traversal：<https://www.youtube.com/watch?v=-BDmzuvm778>
- **机构：** CMU / LeCAR Lab
- **核查日期：** 2026-07-22
- **一句话说明：** HDMI 项目页展示从单目 RGB 人类 HOI 视频到 Unitree G1 全身交互策略的结果：67 次连续开门穿越、6 类真实 loco-manip 任务和 14 类仿真任务，并链接官方代码仓库。

## 核心摘录（归纳，非全文）

- 项目页按钮直接链接 arXiv `2509.16757`、YouTube 视频、35 min door traversal 视频、`LeCAR-Lab/HDMI` 代码仓。
- 摘要强调三段管线：从 unconstrained RGB 视频抽取并重定向人体/物体轨迹；训练 robot-object co-tracking RL policy；零样本部署到 Unitree G1。
- 项目页列出 simulation tasks：Push Door with Hand、Carry and Place Bread Box、Push Box、Move Foam Mats、Roll Ball with Hand、Topple 等。
