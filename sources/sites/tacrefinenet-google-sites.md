## 站点要点（核查）

- **定位：** 多指指尖压阻触觉引导的 **抓取末段局部精修**；三类边缘突出几何（板 / 盘 / 杆）。
- **导航：** Website / Video / Paper / Dataset / Code。
  - **Paper** → <https://arxiv.org/abs/2509.25746>
  - **Code** → <https://github.com/NoneJou072/tacrefinenet>（截至入库日仓库为空）
  - **Dataset** → 当前与 Paper 同链（arXiv abs），**未见独立数据集 URL**
- **演示分区：** 触觉传感器与仿真、任意手内位姿精修、长程扰动跟踪、未见物体泛化（以页内视频为准）。
- **叙事偏差：** 页首 Abstract 仍写「仿真预训练 + 少量真机微调」；与 **arXiv v2**「全程仿真训练、零样本上真机」不一致——**以论文 v2 为准**。
