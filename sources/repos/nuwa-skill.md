## README 要点（归纳）

- **定位：** *「你想蒸馏的下一个员工，何必是同事」* — 蒸馏乔布斯、芒格、费曼、马斯克、Karpathy 等 **各领域最强大脑** 的思维方式。
- **五层提取：** 怎么说话（表达 DNA）→ 怎么想（心智模型）→ 怎么判断（决策启发式）→ 什么不做（反模式）→ 知道局限（诚实边界）。
- **四步工作流：**
  1. **六路并行采集** — 著作、播客/访谈、社交媒体、批评者、决策记录、人生时间线
  2. **三重验证提炼** — 跨 2+ 领域出现、对新问题有预测力、有排他性（非常识）
  3. **构建 Skill** — 3–7 心智模型 + 5–10 启发式 + 表达 DNA + 边界 → `SKILL.md`
  4. **质量验证** — 已知问题方向一致 + 未知问题表现适度不确定
- **已蒸馏：** 14 人物 + 1 主题（X 导师）；独立仓库如 paul-graham-skill、karpathy-skill、steve-jobs-skill 等；官方 skill 经双 agent 盲测保真度 **全员 A 级（≥85）**。
- **诚实边界：** 蒸馏不了直觉、捕捉不了突变、公开表达 ≠ 真实想法。
- **生态：** [darwin-skill](https://github.com/alchaincyf/darwin-skill) 让 skill 持续进化；[cangjie-skill](https://github.com/kangarooking/cangjie-skill) 蒸馏书/长视频方法论。
- **协议：** [Agent Skills](https://agentskills.io)；MIT。
- **贡献规则：** `SKILL.md` 核心不接受外部 PR；社区人物 skill 走 [COMMUNITY.md](https://github.com/alchaincyf/nuwa-skill/blob/main/COMMUNITY.md) 索引。
