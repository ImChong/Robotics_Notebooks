# Orca Hand（虎鲸巧手）

## 一句话定义

**Orca Hand** 是一套面向 **仿生灵巧手** 复刻与二次开发的开源硬件/软件栈：**[orcahand.com](https://www.orcahand.com/)** 提供 CAD、BOM、固件与组装调试说明；源码入口在 GitHub **[orcahand](https://github.com/orcahand)**。

## 为什么重要

- **操作侧硬件入口**：与全身人形解耦后，仍可单独研究 **力控抓取、触觉与遥操作映射**。
- **可复刻 BOM 叙事**：适合作为 **课程/竞赛** 级灵巧手项目的上游引用。
