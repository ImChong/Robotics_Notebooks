## 站点要点

### 定位（首页）

- 自称全球领先的 **多旋翼飞控软件**，FPV **竞速 / 自由式** 社区首选。
- 强调：**性能与精度**、前沿特性、可靠性、广泛飞控硬件支持。
- 开源：源码在 [GitHub](https://github.com/betaflight/betaflight)，GPLv3。

### 核心能力（官网摘要）

| 能力 | 说明 |
|------|------|
| **OSD** | 内置屏幕显示，拖拽配置电量、速度、高度、返航方向等进 FPV 图传 |
| **安全** | 错误配置告警与解锁拦截、失控保护（failsafe） |
| **飞行动力学** | 低延迟扰动响应、摇杆跟踪精度、数字信号处理优化 |
| **硬件** | 多厂商飞控板；Partner 计划由开发团队推荐硬件 |
| **社区** | Facebook、Discord、YouTube 教程生态 |
