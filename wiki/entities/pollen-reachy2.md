# Pollen Reachy2

## 一句话定义

**Reachy2** 是 **Pollen Robotics** 推出的 **开源模块化移动人形**：强调 **仿人双臂、全向移动、VR 遥操作** 与 **软硬件全开放**；主线文档在 **[docs.pollen-robotics.com](https://docs.pollen-robotics.com/)**，代码与示例在 GitHub **[pollen-robotics](https://github.com/pollen-robotics?q=reachy2&type=all&language=&sort=)**，模型与演示空间在 **[Hugging Face · pollen-robotics](https://huggingface.co/pollen-robotics)**。

## 为什么重要

- **具身 AI 研究栈**：同时覆盖 **真机 SDK、仿真、遥操作与 HF 资产**，适合作为 **VLA / 模仿学习** 的整机实验入口之一。
- **与移动操作耦合**：底盘 + 双臂使任务定义天然落在 **loco-manipulation** 交集。
