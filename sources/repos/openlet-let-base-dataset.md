# OpenLET/LET-Base-Dataset

> 来源归档

- **标题：** LET 基础操作数据集（LET-Base-Dataset）
- **类型：** repo
- **代码：** <https://atomgit.com/OpenLET/LET-Base-Dataset>
- **社区页：** <https://openlet.openatom.tech/>
- **采集本体：** 乐聚 **Kuavo 4 Pro**、**Kuavo 5W**
- **入库日期：** 2026-07-17
- **一句话说明：** OpenLET 旗舰发布之一：基于沉浸式 VR 与全身增量遥操采集的 **轮臂基础操作** 真机数据，覆盖抓/拿/放等高频技能，面向工业分拣、搬运与上下料标准化场景。

## 为何值得保留

- **全尺寸人形真机小时：** 与互联网人视频或仿真轨迹不同，直接服务 **VLA / IL 后训练** 与工业任务 benchmark。
- **与工具链闭环：** 同生态 [kuavo-manip-open](https://atomgit.com/OpenLET/kuavo-manip-open) 提供 LeRobot 格式转换与真机部署范例。
