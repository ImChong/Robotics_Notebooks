## 核查结论（步骤 2.5）

- 项目页（画廊）与主仓 **均已公开可运行代码**：技能仓提供 `SKILL.md` + `forge/` 脚本；画廊仓提供生成出的 TypeScript demo 源。
- **非** 摄影测量/权重下载式 image-to-3D 服务；产物是 **diffable TS + JSON spec**。
- 与机器人仿真资产（URDF/碰撞/惯性）无直接导出承诺——选型时勿与 Articraft / PhysForge 等同。
