# PIE: Parkour with Implicit-Explicit Learning Framework for Legged Robots（arXiv:2408.13740）

- **标题：** PIE: Parkour with Implicit-Explicit Learning Framework for Legged Robots
- **作者：** Shixin Luo, Songbo Li, Ruiqi Yu, Zhicheng Wang, Jun Wu, Qiuguo Zhu（浙江大学）
- **发表：** IEEE Robotics and Automation Letters, Vol. 9, No. 11, November 2024
- **链接：** <https://arxiv.org/abs/2408.13740> · DOI <https://doi.org/10.1109/LRA.2024.3459797>
- **一句话说明：** 单阶段端到端跑酷框架：深度图 + 本体历史经多头自编码器做**双层隐式–显式估计**（高度图/速度/足端离地 + 隐式环境表征），低成本四足在挑战地形零样本 sim-to-real 跑酷。

## 核心机制

- **显式头：** 重建高度图、估计基座速度、足端离地高度等可解释物理量。
- **隐式头：** VAE/GRU 记忆压缩难以显式建模的环境因素（摩擦、材质、动态障碍等）。
- **训练：** 仿真中单阶段 PPO + estimator 回归损失；Actor 部署仅用深度 + 本体。
- **对比：** 相对两阶段 teacher–student 视觉跑酷（Extreme Parkour 系）与盲走 DreamWaQ，强调**单阶段**与**不可靠深度相机**下的鲁棒性。
