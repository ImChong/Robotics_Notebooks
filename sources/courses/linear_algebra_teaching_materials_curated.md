# 线性代数优秀教学材料 · 策展索引

> 来源归档（多条外部资源的策展清单，非单本书全文）

- **类型：** course / textbook（策展）
- **入库日期：** 2026-05-31
- **一句话说明：** 与 [Georgia Tech ILA](gatech_interactive_linear_algebra.md)、[Axler LADR4e](axler_linear_algebra_done_right_4e.md) 并列的**互补**线性代数入口，服务本库 [运动控制 L0](../../roadmap/motion-control.md) 与机器人矩阵语言打底。

## 主推荐三角（本 ingest 核心）

| 材料 | 链接 | 角色 |
|------|------|------|
| **Interactive Linear Algebra** | <https://textbooks.math.gatech.edu/ila/> | 交互式、几何直觉 + 计算；见 [gatech_interactive_linear_algebra.md](gatech_interactive_linear_algebra.md) |
| **Linear Algebra Done Right 4e** | <https://linear.axler.net/LADR4e.pdf> | 向量空间 / 线性映射公理化；见 [axler_linear_algebra_done_right_4e.md](axler_linear_algebra_done_right_4e.md) |
| **Essence of Linear Algebra（3Blue1Brown）** | <https://www.3blue1brown.com/topics/linear-algebra> | 短视频建立几何直觉；适合 L0 第一周 |

## 扩展推荐（按学习风格）

| 材料 | 链接 | 适合谁 |
|------|------|--------|
| **MIT 18.06（Gilbert Strang）** | <https://ocw.mit.edu/courses/18-06-linear-algebra-spring-2010/> | 喜欢讲座 + 四大子空间叙事；与 ILA 主题高度重叠 |
| **Introduction to Linear Algebra（Strang 教材站）** | <https://math.mit.edu/~gs/linearalgebra/> | 需要系统习题与工程应用例题 |
| **Numerical Linear Algebra（Trefethen & Bau）** | 教材 PDF 常由机构课程分发；经典参考 *Numerical Linear Algebra*, SIAM | 已会矩阵语言、要做**数值 IK / 最小二乘 / QR** 时精读 |
| **Immersive Math（线性代数卷）** | <https://immersivemath.com/ila/index.html> | 另一套交互式几何导向教材，作 ILA 补充 |

## 为什么值得保留

- 运动控制路线 L0 需要**可维护、可交叉引用**的教材入口，而不是散落的外链。
- 不同材料覆盖同一知识的不同侧面：**几何（3b1b / ILA）→ 公理（LADR）→ 工程四大子空间（Strang）→ 数值（Trefethen）**。
