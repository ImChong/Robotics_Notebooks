# Robotics Notebooks Index

这是 `Robotics_Notebooks` 的知识索引首页。

当前项目从“资源型思维导图”重构为“机器人研究与工程知识库”。

## 项目结构

- `sources/`：原始资料
- `wiki/`：结构化知识页
- `schema/`：知识库规则
- `exports/`：面向网页渲染的导出层
- `log.md`：重构与更新日志

## 当前优先方向

第一阶段重点是搭好知识骨架，而不是重写网页。

### 重点主题
- Humanoid Control
- Reinforcement Learning
- Imitation Learning
- Sim2Real
- Locomotion
- Manipulation
