## 为什么值得保留

- **Lab 与公司的分工**：`roboparty.com` 偏公司与整机产品；`lab.roboparty.com` 偏 **研发基础设施与学术/工程产出**，是 [Party OS](../repos/party_os.md) 的一手门户。
- **Party OS 四支柱**：Locomotion / Perception / Manipulation / Agent 的官方中文表述，可直接支撑技术地图与 [party-os](../../wiki/entities/party-os.md) 页。
- **成果与荣誉**：TeCH 成果页、AGIBOT Challenge Top 3、EAI OpenSource 10 等条目可与公众号 ingest 交叉验证。

## 门户摘录（2026-07 抓取）

### Lab 定位

> RoboParty Lab 是一个纯粹的技术探索之地……AGI 将会诞生于经历过 RoboParty Lab 的人之手。

**价值公式（站点多处重复）**：

`idea + infra + Assistant Professor + 平台 + 3~6 个月` → 让有能力、有技术、有想法的人被快速看到；本体、数据集、sim2real、论文叙事与影响力由 Lab 基础设施承接。

### Party OS 四方向

| 方向 | 英文 | 官方重点 |
|------|------|----------|
| 基础运动控制 | Locomotion | 数据生成、重定向、训练框架、Sim2Real/Real2Sim、BFM 基础设施 |
| 感知交互 | Perception | 环境感知、HSI、HOI；复杂真实环境连续交互 |
| 全身协调控制与操作 | Manipulation | BFM + VLA + World Model；全身协调与复杂操作 |
| 智能体系统 | Agent | Agent + Skills；任务规划、自主执行与技能组合 |
