# OmniTacTune 项目页归档

> 来源：<https://colinyu1.github.io/omnitactune-site/>（ingest 快照）

- **标题：** OmniTacTune: Policy-Agnostic Real-World RL for Tactile Residual Adaptation of Visual Policies
- **类型：** project-site
- **论文：** arXiv:2607.03723
- **入库日期：** 2026-07-08

## 站点要点（与论文一致）

- **定位：** 冻结视觉策略 + **触觉残差** 真机 RL；**无需离线触觉演示**。
- **效率：** 四接触丰富任务 **40–80 min** 在线练习，基策略 **5–40% → 85–100%**。
- **基策略：** Human Flow、ACT、DP、π₀.₅；数据源含 **人视频** 与 **VR 遥操**。
- **触觉表示：** AnyTouch2、Sparsh、T3、低维 markers 均可。
- **任务演示视频：** Peg-in-Hole、Charger Insertion、Cap Opening、Box Opening；含 **40 min 一镜到底** 在线 RL 训练片段。
