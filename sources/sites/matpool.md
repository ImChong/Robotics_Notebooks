# 矩池云（Matpool）

> 来源归档

- **标题：** 矩池云 — 专注于人工智能领域的云服务商
- **类型：** site（国内 GPU 云算力平台）
- **来源：** 矩池云
- **链接：** https://www.matpool.com/
- **入库日期：** 2026-07-02
- **一句话说明：** 面向高校与科研团队的 GPU 云算力平台，按分钟计费，提供算力市场、AI 功能岛与专有云/私有云方案；卡型覆盖 A2000 至 A100，默认数据盘较大（150–800GB 因机型而异）。

## 为什么值得保留

- **高校渗透率高**：官网称服务 300+ 高校及研究机构，适合学生/实验室首次租卡。
- **大容量免费磁盘**：4090 实例示例含 200GB 硬盘、A100 可达 800GB，利于数据集与 checkpoint 本地缓存。
- **按分钟计费**：比纯按小时平台更细粒度控费；另有 AI 功能岛按分计费。

## 平台要点（官网 2026-07 摘录）

| 维度 | 要点 |
|------|------|
| **产品** | 算力市场（GPU 主机）、AI 功能岛、专有云/私有云/软硬一体大模型方案 |
| **计费** | 按分钟计费；新人体验金等活动 |
| **稳定性** | 宣称 99.9% 机器稳定性、7×24 支持 |
| **典型卡型** | A2000 12G、RTX 3090 24G、RTX 4090 24G、A100 80G 等 |
| **场景** | 模型训练、图形渲染、高校 AI 实训、汽车 CAE/自动驾驶仿真（行业方案页） |
