# EGM: Efficiently Learning General Motion Tracking Policy for High Dynamic Humanoid Whole-Body Control

- **URL（PDF）**: https://arxiv.org/pdf/2512.19043
- **URL（HTML / abs）**: https://arxiv.org/abs/2512.19043
- **Authors**: Chao Yang, Yingkai Sun, Peng Ye, Xin Chen, Chong Yu, Tao Chen（Shanghai Innovation Institute、复旦大学、CUHK、ByteDance 等）
- **Date**: 2025-12（arXiv 预印本）
- **Tags**: #humanoid #motion-tracking #rl #ppo #dagger #moe #curriculum-sampling #unitree-g1

## 核心摘要

**EGM（Efficient General Mimic）** 面向「**单一策略**跟踪**多段、多分布**人体参考运动」的人形全身控制：在 **动捕数据冗余 / 不平衡** 与 **高动态动作跟踪不稳** 两类痛点上，用 **Bin 级跨动作课程自适应采样（BCCAS）**、**复合解耦混合专家（CDMoE）**、**数据策展** 与 **三阶段教师–学生训练** 组合，报告在仅 **4.08 小时** 训练动作上训练即可在 **49.25 小时** 测试动作上泛化，并在常规与高动态场景上优于若干 **通用 tracking** 基线（文中与 GMT 等对照）。

### 技术路线（摘录）

1. **问题与平台**：**目标条件 MDP**；**Unitree G1**，文中控制 **23 DoF**（腕部 3×2 固定）；动作为 **目标关节角**，经 **PD** 成扭矩。
2. **BCCAS 采样**：将全部动作切成 **1 s motion bin** 并赋全局 ID；训练周期性 **跨 bin 全局采样**，以 bin 为起点再 **0–5 s 随机前向偏移**，截取至多 **20 s** 跟踪片段，缓解「整段 clip 采样」下 **长短序列与单 clip 内动作异质性** 的不公平；用 **复合跟踪误差（位姿/速度等多指标加权）** 的 **EMA** 归一化后转成各 bin 的采样概率；再引入 **均匀采样比例 + 幂次平滑** 的 **采样课程**，早期偏探索、后期概率分布变尖以偏重高误差区域。
3. **CDMoE 策略结构**：在 MoE/OMoE 思路上，**上半身 8 DoF / 下半身 15 DoF 分组专家**；每组内 **Gram–Schmidt 正交专家 + top‑k 门控加权** 抽取 **运动专用特征**，再与 **不经正交化的共享专家** 输出相加得组合特征，经统一 **输出头** 得动作，从而显式分离 **通用（如平衡）** 与 **专用动态** 特征并避免上下身抢同一组专家容量。
4. **数据**：**AMASS** 与 **LAFAN1** 经 **形状–运动两阶段重定向**；剔除不可实现/重定向失败片段；对比 **人工精选小集 𝒟_small（674 条，4.08 h）** 与 **规则预筛 + 用初训策略打分再剪枝的大集 𝒟_large（10 393 条，27.47 h）**，实验结论为 **小集在泛化与精度上更好**——强调 **「通用 motion tracking」里数据质量与多样性优先于单纯堆小时数**。
5. **训练**：**教师** 两阶段 **PPO**（Stage1 基础跟踪 → Stage2 **域随机化 + 复杂地形 + 更细跟踪项/更强惩罚**），**学生** 用 **DAgger** 蒸馏教师，学生侧用 **历史编码器** 补偿去掉特权观测后的信息损失。
