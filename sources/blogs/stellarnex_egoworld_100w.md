## 核心摘录（归纳，非全文）

### 规模与多样性口径（页面宣称）

- **100 万+** 第一人称操作视频；累计 **10,000+** 小时；**3,000+** 参与者。
- **8** 类真实场景；估计 **20 万+** 互不重复视觉背景。
- 分层抽样估计：**2,800+** 物体语义类、**190+** 动作类；细粒度物体变体 **80,000+**、动作实例 **57 万+**。
- “100W” 的 W 取中文「万」，表示百万量级。

### 组织方式：Task / Sub-Task + 四维结构

- 视频级自然语言 **Task**，可拆 **1–3** 个 **Sub-Task**（最小操作单元）。
- 每个 Sub-Task 锚定四维：**Scene / Object / Action / Handedness**。
- 场景占比示例（长尾）：家庭 ~34%、办公 ~17%、工业 ~15%、零售 ~15%、餐饮 ~6%、物流 ~6%、公共 ~5%、其他 ~2%。
- 动作可归并为 7 个语义组（抓取放置、表面操作、状态改变、精细操控、空间移动、收纳组织、工具化检测等）。

### 采集与处理

- 统一头戴 **单目** 第一人称；分辨率归一化、时间轴规范化、元数据绑定；场景分类、Task 撰写、中英双语、质控。
- 官网另宣传双目头戴硬件（规格表）与 3D 姿态重建引擎——本 blog 正文聚焦数据集叙事。

### 保真度示例（HaMeR）

- 相机轨迹 MAE **0.0212 m** / RMSE **0.0276 m**
- 相机–腕距离（双手）MAE **0.013 m** / RMSE **0.0168 m**
- 手部关节 MPJPE **0.0126 m**；指尖均值误差 **0.0139 m**

### 定位对比（页面叙述）

- 相对 EPIC-KITCHENS-100、Ego4D、EgoMimic、EgoDex：更强调 **真实世界多场景 + 大规模 + 结构化覆盖**，服务组合迁移与诊断性泛化评测，而非单一厨房细粒度理解或受控高精度采集。

### 应用与合作

- 面向 VLA / 视频世界模型预训练、模仿学习、四维控制变量诊断评测、组合迁移、自中心感知与重建。
- **三种合作档：** 个人体验级 / 高校研究级 / 企业共创与商用授权；申请走飞书表单（主页「合作需求登记」）。
