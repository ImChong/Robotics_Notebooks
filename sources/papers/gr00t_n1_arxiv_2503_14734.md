# GR00T N1: An Open Foundation Model for Generalist Humanoid Robots（arXiv:2503.14734）

> 来源归档（ingest · arXiv 全文 + NVIDIA 官方白皮书）

- **标题：** GR00T N1: An Open Foundation Model for Generalist Humanoid Robots
- **类型：** paper / humanoid / VLA / foundation model
- **arXiv：** <https://arxiv.org/abs/2503.14734>（HTML：<https://arxiv.org/html/2503.14734v2>）
- **PDF：** <https://arxiv.org/pdf/2503.14734>
- **官方白皮书：** <https://d1qx31qr3h6wln.cloudfront.net/publications/GR00T_1_Whitepaper.pdf>（NVIDIA，2025-03-18）
- **代码：** <https://github.com/NVIDIA/Isaac-GR00T>
- **数据集：** <https://huggingface.co/datasets/nvidia/PhysicalAI-Robotics-GR00T-X-Embodiment-Sim>
- **机构：** NVIDIA（GEAR Lab；研究负责人 Linxi Fan、Yuke Zhu）
- **入库日期：** 2026-07-02
- **一句话说明：** 开源人形通才 VLA 基础模型：Eagle-2 VLM（System 2，10Hz）+ DiT flow-matching 动作头（System 1，120Hz）；以数据金字塔（人类视频 / 仿真与神经合成 / 真机）联合预训练，公开 GR00T-N1-2B（2.2B 参数）并在 Fourier GR-1 真机双臂操作验证高数据效率。

## 核心摘录（策展，非全文）

1. **双系统 VLA 架构：** System 2 为 Eagle-2 VLM（SmolLM2 + SigLIP-2，224×224，取第 12 层隐状态，L40 上约 10Hz）；System 2 输出经 cross-attention 条件化 DiT flow-matching 动作模块（System 1），以 embodiment-specific MLP 编码/解码可变维状态与动作，action chunk $H=16$，推理 $K=4$ 步去噪，L40 bf16 采样 16 步动作约 63.9ms（≈120Hz 闭环）。
   - **对 wiki 的映射：** [paper-hrl-stack-34-gr00t_n1](../../wiki/entities/paper-hrl-stack-34-gr00t_n1.md)、[VLA](../../wiki/methods/vla.md)

2. **数据金字塔与共训练：** 底层为人类 egocentric 视频（Ego4D、Ego-Exo4D、HOI4D 等）与 VLM 预训练 web 数据；中层为 DexMimicGen 仿真轨迹（预训练约 54 万条 / 6500 小时，11 小时生成）与 image-to-video 神经轨迹（88 小时遥操作扩至 827 小时，约 10×）；顶层为 GR-1 真机遥操作、Open X-Embodiment 与 AgiBot-Alpha（当时 14 万条）等。无动作标签视频用 **latent action（LAPA embodiment）** 与 **IDM 伪动作** 统一进 flow-matching 目标。
   - **对 wiki 的映射：** [paper-hrl-stack-34-gr00t_n1](../../wiki/entities/paper-hrl-stack-34-gr00t_n1.md)、[Foundation Policy](../../wiki/concepts/foundation-policy.md)

3. **训练规模与开源：** 预训练最多 1024×H100，GR00T-N1-2B 预训练约 5 万 H100 GPU 小时；后训练按具身冻结 VLM 语言部分、微调其余模块。公开 checkpoint、训练数据与仿真 benchmark（RoboCasa / DexMG / GR-1 Tabletop）。
   - **对 wiki 的映射：** [gr00t-wholebodycontrol](../../wiki/entities/gr00t-wholebodycontrol.md)

4. **仿真评测（每任务 100 条 demo 后训练）：** RoboCasa 32.1%、DexMG 66.5%、GR-1 Tabletop 50.0%，平均 45.0%；优于 BC-Transformer（26.4%）与 Diffusion Policy（33.4%）。
   - **对 wiki 的映射：** [paper-hrl-stack-34-gr00t_n1](../../wiki/entities/paper-hrl-stack-34-gr00t_n1.md)

5. **真机评测（Fourier GR-1，语言条件双臂桌面操作）：** 预训练 checkpoint 在双手递送（76.6%）与未见物体放入未见容器（73.3%）表现良好；全量后训练平均成功率 **76.8%**（Diffusion Policy 46.4%）；仅 10% 数据后训练仍达 **42.6%**（DP 全量 46.4%，DP 10% 仅 10.2%）。神经轨迹共训练在 RoboCasa 低数据区平均 +4.2%～+8.8%。
   - **对 wiki 的映射：** [paper-hrl-stack-34-gr00t_n1](../../wiki/entities/paper-hrl-stack-34-gr00t_n1.md)

6. **局限（论文 §4.6）：** 当前聚焦短视界桌面操作；长视界 loco-manipulation、更强 VLM 空间推理、物理一致合成数据与架构仍待扩展。
   - **对 wiki 的映射：** [Loco-Manipulation 任务页](../../wiki/tasks/loco-manipulation.md)
