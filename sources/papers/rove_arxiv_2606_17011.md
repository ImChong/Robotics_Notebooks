# ROVE: Unlocking Human Interventions for Humanoid Manipulation via Reinforcement Learning

> 来源归档（ingest）

- **标题：** ROVE: Unlocking Human Interventions for Humanoid Manipulation via Reinforcement Learning
- **类型：** paper
- **来源：** arXiv abs / PDF；项目页交叉核对
- **原始链接：**
  - <https://arxiv.org/abs/2606.17011>
  - <https://arxiv.org/pdf/2606.17011>
  - <https://xpeng-robotics.github.io/rove/>
- **机构：** XPENG Robotics；复旦大学；香港中文大学；上海交通大学
- **作者：** Wei Xiao, Weiliang Tang, Yuying Ge, Hui Zhou, Yao Mu, Li Zhang, Yixiao Ge（*Equal contribution: Xiao, Tang；†Corresponding: Ge）
- **入库日期：** 2026-06-18
- **一句话说明：** 面向 **人形 VLA 后训练** 的 **人机闭环 RL** 框架：全身 MoCap 遥操作在 VLA 近失败时接管，将轨迹分解为 rollout / adaptation / recovery 三阶段；以 **Optimistic Value Estimation（OVE）** 与 **跨 embodiment 人类视频** 训练状态价值 critic，再用 **advantage conditioning** 从混合质量经验中提取更强 VLA 策略；真机接触丰富与精细操作任务上优于 SFT 与 HG-DAgger / Filtered BC / RECAP，并随 rollout–intervention 迭代持续提升。

## 核心论文摘录（MVP）

### 1) 问题：人形干预数据并非专家示范

- **链接：** <https://arxiv.org/abs/2606.17011> §1
- **摘录要点：** VLA 在 **平行夹爪机械臂** 上的 post-training 已证明部署经验可超越离线 SFT；扩展到 **全身 + 灵巧手人形** 后，**遥操作接管** 本身往往 **次优**——操作员需适应当前构型、犹豫、回撤、受重定向与关节限位影响。传统 HIL-SERL / HG-DAgger / RECAP 常假设干预即 **专家纠正**，在人形场景会把 **adaptation 噪声** 一并模仿进策略。
- **对 wiki 的映射：**
  - [ROVE（论文实体）](../../wiki/entities/paper-rove-humanoid-vla-intervention.md) — 问题定义与 teleoperation gap
  - [Teleoperation](../../wiki/tasks/teleoperation.md) — 人形全身干预的系统难点

### 2) 人机闭环采集：三阶段 episode 分解

- **链接：** <https://xpeng-robotics.github.io/rove/>（Method）
- **摘录要点：**
  - Episode 以 **VLA 自主 rollout** 开始；监督员预判失败时暂停策略，**MoCap 操作员经 VR** 对齐全身与灵巧手后切换遥操作。
  - 控制器 **command filter** 平滑接管瞬态；后处理去掉暂停间隔。
  - 三阶段：**(1) 自主 VLA rollout** → **(2) intervention–adaptation**（对齐、犹豫）→ **(3) intervention–recovery & completion**。
  - 奖励：成功终态 0；自主失败或 **adaptation 结束时刻** $t_r$ 赋 $C_{\mathrm{fail}}$（保守边界，避免把对齐行为标成 recovery）；其余步 $-1$；$\gamma=1$，靠步惩罚隐式编码时间成本。
- **对 wiki 的映射：**
  - [ROVE](../../wiki/entities/paper-rove-humanoid-vla-intervention.md) — Mermaid 流程与阶段感知标注

### 3) Optimistic Value Estimation（OVE）与跨 embodiment 价值学习

- **链接：** <https://arxiv.org/abs/2606.17011> §3.2
- **摘录要点：**
  - Critic **预训练**：大规模机器人 + egocentric 人类演示上 **Monte-Carlo return 回归**。
  - **任务微调**：机器人 rollout、干预轨迹 + **人类经验视频**（动作空间不可比，仅提供进度/恢复监督）。
  - **OVE**：$H$ 步 TD bootstrap + **expectile regression**（$\tau\to 1$ 更乐观），估计 **in-distribution 高价值恢复** 而非平均回报；学 **状态价值 $V(s)$** 而非 $Q(s,a)$，避免跨 embodiment 动作不可比与 OOD 过估计。
  - Actor：**advantage conditioning** 对 action chunk 二值化高/低价值，强调高价值行为而非均匀模仿全轨迹。
- **对 wiki 的映射：**
  - [ROVE](../../wiki/entities/paper-rove-humanoid-vla-intervention.md) — OVE 与 critic 数据配方
  - [Online vs Offline RL](../../wiki/comparisons/online-vs-offline-rl.md) — 部署经验后训练语境

### 4) 真机实验与迭代改进

- **链接：** <https://xpeng-robotics.github.io/rove/>（Results）
- **摘录要点：**
  - **平台：** IRON-R01-1.11 人形，50-D 本体状态/动作（身体关节 + 灵巧手）。
  - **任务：** 擦白板（接触丰富）、面包入吐司机（精细）；附加 **螺丝安装**（毫米级精度）。
  - **对比：** 仅演示 SFT、HG-DAgger、Filtered BC、RECAP；ROVE 平均成功率最佳；HG-DAgger 甚至低于演示策略且策略犹豫——直接模仿干预的反面证据。
  - **三轮迭代：** 擦白板 45.0%→80.0%；面包 56.7%→86.7%；部署可见 **重试插入、补擦遗漏区域** 等显式恢复行为。
  - **价值消融：** 人类视频降低对未完成擦除状态的过乐观；OVE 相对 MC 在失败/恢复段给出更清晰负 advantage 区。
- **对 wiki 的映射：**
  - [ROVE](../../wiki/entities/paper-rove-humanoid-vla-intervention.md) — 实验与基线对照表
  - [Manipulation](../../wiki/tasks/manipulation.md) — 接触丰富与精细操作语境

### 5) 局限与未来方向

- **链接：** <https://arxiv.org/abs/2606.17011> §7
- **摘录要点：** 人类视频目前 **仅用于 value** 未直接训 policy；缺腕部相机/触觉；尚未扩展 loco-manipulation；主框架为 **离线/迭代离线 RL**，在线探索仍待研究。
