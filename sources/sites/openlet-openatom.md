# OpenLET 具身智能开源数据集社区（openlet.openatom.tech）

> 来源归档

- **标题：** OpenLET 具身智能开源数据集社区
- **类型：** site
- **官方入口：** <https://openlet.openatom.tech/>
- **运营方：** [开放原子开源基金会](https://www.openatom.org/) 孵化；**乐聚机器人**牵头建设运营
- **联系：** weiziyu@lejurobot.com
- **入库日期：** 2026-07-17
- **一句话说明：** 面向具身智能的 **真机数据开源枢纽**：发布 **LET** 系列数据集（全身运控、灵巧手操作、轮臂基础操作等，合计 **>60,000 分钟** 高质量 Kuavo 真机数据），并聚合 CRAIC / REAL-I / 挑战杯等赛事与 LeRobot 复现教程。

## 开源仓库（AtomGit，截至 2026-07-17 首页趋势）

| 仓库 | 说明 |
|------|------|
| [OpenLET/LET-Base-Dataset](https://atomgit.com/OpenLET/LET-Base-Dataset) | 轮臂基础操作：Kuavo 4 Pro / 5W + VR 全身增量遥操，抓拿放等工业分拣场景 |
| [OpenLET/LET-Dex-Dataset](https://atomgit.com/OpenLET/LET-Dex-Dataset) | 灵巧手操作：外骨骼 + 力触觉灵巧手，捏扣握等高维触觉动捕 |
| [OpenLET/LET-Body-Dataset](https://atomgit.com/OpenLET/LET-Body-Dataset) | 全身运控：VR + 全身动捕重定向，行走/下蹲/转腰/双足跟随 |
| [OpenLET/kuavo-manip-open](https://atomgit.com/OpenLET/kuavo-manip-open) | 基于 LeRobot：rosbag→parquet、IL 训练、仿真与真机部署示例 |
| [OpenLET/kuavo_data_challenge](https://atomgit.com/OpenLET/kuavo_data_challenge) | 数据挑战赛相关资源 |

## 社区栏目（策展）

- **LET 数据集** — 工业 / 商零 / 家庭三大领域真机小时
- **REAL-I 具身智能挑战赛** — ICRA 背书，工业真任务评测
- **乐聚具身智能操作任务挑战赛 & 创业启航营** — 「真实任务 + 开源数据 + 真机评测」
- **CRAIC 人形机器人挑战赛** — Kuavo 教程与备赛资源
