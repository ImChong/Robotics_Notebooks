# Mixamo（Adobe 在线角色与动画库）

- **标题**: Mixamo
- **类型**: commercial-web-service / character-animation
- **官网**: https://www.mixamo.com
- **主体**: Adobe 旗下在线服务（页面文案与「Do more with 3D&AR tools from Adobe」交叉推广 Aero / Dimension / Substance）
- **收录日期**: 2026-05-15

## 一句话摘要

面向影视、游戏与交互内容的 **绑定角色** 与 **大量全身动画** 浏览与下载平台，支持上传自定义网格做 **自动骨骼绑定**，并导出多种常见 3D 格式；**商业服务**，使用条款与导出许可以 Adobe 官方为准。

## 为何值得保留

- **快速原型**：在尚无自采 MoCap 时，可用标准化骨骼动画为游戏引擎或仿真里的人类参考轨迹做 **视觉与节奏** 占位。
- **与机器人研究的张力**：动画来自专业演员动捕，但经角色绑定与美术管线，**不等同于**可直接用于严肃动力学辨识的「原始 MoCap」；与 [ZEST](../../wiki/methods/zest.md) 等讨论「异构人类运动源」时形成对照。
- **合规独立评估**：企业或开源项目分发衍生资产前，应核对 Adobe 许可与项目授权是否允许再分发。

## 站点公开产品叙事（编译自首页）

- **即用角色**：多种风格预设角色，带贴图与骨骼。
- **自动绑定（Auto-Rigging）**：上传自定义人形网格生成骨架（页面宣称可针对移动平台做 rig 选项）。
- **动捕动画库**：大量全身动作，可预览并迁移到所选角色。
- **多格式导出**：面向 motion graphics、游戏、影视等管线（具体格式以站点导出 UI 为准）。
