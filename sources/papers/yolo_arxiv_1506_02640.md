# YOLO v1：统一实时目标检测（arXiv:1506.02640）

> 论文来源归档（ingest）

- **标题：** You Only Look Once: Unified, Real-Time Object Detection
- **作者：** Joseph Redmon, Santosh Divvala, Ross Girshick, Ali Farhadi（UW / Allen AI / FAIR）
- **类型：** paper / computer-vision / object-detection / real-time
- **arXiv：** <https://arxiv.org/abs/1506.02640> · PDF：<https://arxiv.org/pdf/1506.02640.pdf>
- **项目页：** <http://pjreddie.com/yolo/>
- **训练框架：** [Darknet](https://github.com/pjreddie/darknet)
- **入库日期：** 2026-06-06
- **一句话说明：** 将检测 **重构为单次回归**：整图经 **单一 CNN** 直接输出 **S×S 网格上的边界框 + 类别概率**，在 Titan X 上达 **45 FPS**，开启 **端到端实时检测** 范式，并深刻影响后续机器人机载感知栈（YOLO 系列至今）。

## 核心摘录（面向 wiki 编译）

### 1) 统一检测 = 回归问题

- **要点：** 摒弃 **滑动窗口 / 区域提议 + 分类器** 的多阶段管线；一次前向同时预测 **所有类别的所有框**；全图训练使模型 **全局推理上下文**，背景误报显著少于 Fast R-CNN。
- **对 wiki 的映射：** [`wiki/entities/paper-yolo-unified-realtime-detection.md`](../../wiki/entities/paper-yolo-unified-realtime-detection.md)、[`wiki/methods/object-detection.md`](../../wiki/methods/object-detection.md)

### 2) 网格预测与置信度

- **要点：** 输入 resize 至 **448×448**；**S=7** 网格，每格预测 **B=2** 框 + 置信度 $\Pr(\text{Object}) \times \text{IOU}$；每格 **C=20** 条件类概率（VOC）；输出张量 **7×7×30**。
- **对 wiki 的映射：** 同上实体页

### 3) 网络与训练

- **要点：** **24 层卷积 + 2 FC**，结构受 GoogLeNet 启发；ImageNet 预训练前 20 层再 **升分辨率微调**；多部件损失含 $\lambda_{\text{coord}}=5$、$\lambda_{\text{noobj}}=0.5$；**Responsible predictor** 按最高 IOU 分配梯度。
- **对 wiki 的映射：** [`wiki/concepts/vision-backbones.md`](../../wiki/concepts/vision-backbones.md)（预训练骨干语境）

### 4) 速度–精度权衡与局限

- **要点：** 基线 **63.4 mAP @ 45 FPS**；Fast YOLO **52.7 mAP @ 155 FPS**；主要错误来自 **定位** 而非背景；对小目标与密集目标受限（每格仅 2 框、单类）。
- **对 wiki 的映射：** [`wiki/methods/object-detection.md`](../../wiki/methods/object-detection.md)

### 5) 与 R-CNN 族组合

- **要点：** YOLO 重打分 Fast R-CNN 可 **+3.2 mAP**（75.0），因 YOLO 更少背景假阳性。
- **对 wiki 的映射：** [`wiki/methods/object-detection.md`](../../wiki/methods/object-detection.md)
