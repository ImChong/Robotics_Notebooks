## 一句话摘要

清华等联合团队的 **Harness VLA** 官方站点：强调 **冻结 VLA、学习如何使用**——参考种子探索蒸馏原语轨迹、成功规则与失败模型，部署时对扰动场景 **重绑定** 而非重放坐标。

## 公开信息要点（截至入库日 2026-07-22）

- **核心叙事：** Memory-guided agentic manipulation；`vla_act` 负责接触密集相位，解析原语负责 grounding / staging / transport / release / recovery。
- **结果卡片（与论文一致口径）：** LIBERO-Pro **82.4%**（RLinf 冻结基线 50.0%，+32.4 pts 卡片对比）；RoboCasa365 **55.4%**（RLDX-1 30.0%，+25.4）；RoboTwin C2R **58.4%**（π₀.₅ 47.9%，+10.5）。
- **资源链接：** Paper → arXiv PDF；Code → `RLinf/RPent`；Video *coming soon*。
- **机构署名：** Tsinghua / Striding AI / Purdue / CASIA / Infinigence AI / HKUST / Zhongguancun Academy。
