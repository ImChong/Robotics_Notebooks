# Before the Body Moves: Learning Anticipatory Joint Intent for Language-Conditioned Humanoid Control（DAJI）

- **类型**：论文
- **收录日期**：2026-05-19
- **arXiv**：<https://arxiv.org/abs/2605.14417>
- **PDF**：<https://arxiv.org/pdf/2605.14417>
- **项目页**：<https://hxxxz0.github.io/DAJI_PAGE/>
- **代码**：<https://github.com/Hxxxz0/DAJI>
- **机构**：HKUST（广州）、LimX Dynamics、山东大学、JITRI、Data61/CSIRO、格里菲斯大学等

## 一句话

提出 **预期关节意图（anticipatory joint intent）** 作为语言条件人形控制的 **可部署中间接口**：DAJI-Flow 自回归生成低频意图片段，DAJI-Act 结合实时本体感知解码为高频全身动作，替代「上层参考轨迹 + 底层被动追踪」范式。

## 为什么值得保留

- **问题重框**：流式语言全身控制的瓶颈不仅是「听懂指令」，而是 **语义–动力学之间** 是否有一层 **可执行且带未来准备信息** 的表示。
- **与参考轨迹路线的对比**：完整运动参考易与当前重心/接触/动量脱节；过于抽象的技能编码又可能缺少支撑切换与平衡准备信息。
- **系统证据**：消融显示 **去掉未来信息后 60s 闭环成功率从约 87% 跌至约 10%**，强调长程流式指令下 **身体状态不可重置** 的特性。

## 核心摘录（面向 wiki 编译）

### 架构

| 模块 | 角色 |
|------|------|
| **DAJI-Flow** | 根据语言与历史意图，自回归预测未来 **关节意图片段**（低频） |
| **DAJI-Act** | 将意图解码为高频动作；结合实时本体感知；由 **未来感知教师** 蒸馏为可部署扩散策略 |
| **Joint-intent 接口** | 生成输出 **即部署控制接口**，而非仅供显示的 kinematic reference |

### 主要结果（项目页 / 公众号编译口径）

- HumanML3D 风格生成：**成功率约 94.42%**（TextOp 约 90.00%，MotionStreamer 约 87.50%）；单指令 FID 约 0.147。
- BABEL 长时序流式切换：**子序列 FID 约 0.152**（TextOp 约 0.538），转换更连续。
- 部署接口：64 维 MLP 在 CPU 上延迟约 **4.71 ms**，成功率约 **80.8%**；16 维降至约 49.5%。
- **未来信息消融**：60s 成功率约 **87% → 10%**（无未来准备信息时长程闭环崩溃）。

### 与相邻工作的关系（索引级）

- **OmniH2O**：多源输入（VR/RGB/语言等）统一到人形 **身体接口**；DAJI 专注 **语言流式生成 + 预期意图**。
- **TextOp / MotionStreamer**：语言→人形运动生成的对比基线。
- **MDM / ASE / CALM**：语言或任务驱动的人体运动表示谱系；DAJI 更靠近 **真实闭环人形部署**。
