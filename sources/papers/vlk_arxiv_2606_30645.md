# VLK: Learning Humanoid Loco-Manipulation from Synthetic Interactions in Reconstructed Scenes

- **项目页：** <https://vision-language-kinematics.github.io>
- **机构：** Amazon FAR；UC Berkeley；Stanford University；Carnegie Mellon University
- **入库日期：** 2026-07-03
- **更新日期：** 2026-07-22
- **开源状态：** 待发布；项目页 Code 标注 **coming soon**。页面 footer 的 GitHub 链接是项目网页模板源码，不是 VLK 训练/部署代码。
- **一句话说明：** 在 **3DGS 重建场景** 中合成 **48k** 视觉-语言-全身运动学三元组，训练 **VLK** 策略预测 1 秒未来 G1 kinematic trajectory 和 wrist contact labels，经 **SceneBot** tracker 在 **Unitree G1** 上零真实微调部署导航与单物体搬运。

## 摘要要点

- Perception-based humanoid loco-manipulation 缺少同步的 ego image、language command 和 robot-compatible whole-body trajectory。
- VLK 用 3DGS 重建真实室内场景，通过 privileged scene information 合成导航和物体交互轨迹，再 hindsight 渲染 ego RGB。
- 每个环境生成 **48,000** 条 paired trajectories，无需人类示范；策略从 π0.5 初始化。
- 部署时高层预测 G1 whole-body kinematics，低层 whole-body tracker 将其转为真机动作。
- 真机 Unitree G1 在 lab/apartment 场景下完成导航、pick、put、长时程 chaining、布局变化和光照扰动测试。

## 方法要点

- **Scene reconstruction：** iPhone 14 Pro / Polycam 扫描，重建 metric-scale 3DGS；手动标注 semantic 3D boxes 与 walkable regions。
- **Waypoint generation：** 按任务规则采样初始姿态、目标物、walkable waypoints，并自动生成语言模板。
- **Motion synthesis：** navigation model 基于 BONES-SEED；interaction model 基于 OMOMO 经 OmniRetarget 到 G1 后训练条件扩散模型。
- **Interaction conditions：** language instruction、initial humanoid/object state、scene waypoints、object geometry、relative wrist poses、wrist-object contact labels。
- **Egocentric rendering：** Isaac Sim 回放 G1 轨迹，虚拟 ZED 2i 头部相机渲染；对相机、光照和图像做 domain randomization。
- **VLK policy：** 输入 ego RGB、language、当前 G1 state；输出 H=30 frames at 30 Hz 的 future trajectory；SceneBot 50 Hz 跟踪。

## 实验与数字

- **数据规模：** 4 lab layouts + 4 apartment layouts；每环境 4 layouts × 12 modes × 1000 trajectories = **48,000 trajectories**。
- **生成耗时：** 单 L40S 上，一个 layout/mode 的 1000 条 motion synthesis 约 **4 h**，egocentric rendering 约 **8.3 h**。
- **真机评测：** 每个 mode 20 trials，Lab / Apartment 两场景合计 120 real-world trials。
- **真机成功：** Walk To 20/20 与 19/20；Put Floor 20/20 与 20/20；Pick Surface 11/20 与 13/20；Put Surface 8/20 与 15/20。
- **视觉随机化：** walking success 从 no randomization **41%** 提升到 full randomization **90%**。
- **部署延迟：** RTX 5090 VLK inference **31 ms**；end-to-end replan 平均约 **63 ms**。
