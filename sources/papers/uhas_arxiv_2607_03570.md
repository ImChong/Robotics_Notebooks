# Cross-Embodiment Robot Manipulation via a Unified Hand Action Space

> 来源归档（ingest）

- **标题：** Cross-Embodiment Robot Manipulation via a Unified Hand Action Space
- **类型：** paper
- **机构：** 德克萨斯大学达拉斯分校（UT Dallas）IRVL；亚利桑那州立大学（ASU）IRIS Lab
- **原始链接：**
  - <https://arxiv.org/abs/2607.03570>
  - 项目页：<https://irvlutd.github.io/UHAS/>
- **发表：** RSS 2026 第 4 届灵巧操作研讨会（Dexterous Manipulation Workshop）
- **入库日期：** 2026-07-11
- **一句话说明：** 提出 **统一手部动作空间（UHAS）**——把灵巧手动作表示为 **规范球面几何形变**，经 **级联逆运动学（CIK）** 映射到 Allegro / LEAP / Shadow / MANO 等不同运动学结构；在 **手内立方体重定向** 上用 **单一 PPO 策略** 实现多手联合训练、零样本迁移与快速微调，并在真机 LEAP Hand 上完成 sim2real 部署。

## 核心论文摘录（MVP）

### 1) 问题：灵巧手策略绑定 embodiment 专用关节空间

- **链接：** <https://arxiv.org/abs/2607.03570> §1
- **摘录要点：** 大规模机器人学习仍以平行夹爪为主；灵巧手 **运动学结构、动作参数化、控制接口** 差异大，现有系统常 **按手型单独定义动作空间与重训**，跨平台迁移困难。
- **对 wiki 的映射：**
  - [UHAS](../../wiki/methods/uhas-unified-hand-action-space.md) — 把「跨具身灵巧手控制」定位为 **统一几何动作接口** 问题，而非单纯数据堆叠。

### 2) 规范球构建与稠密表面对应

- **链接：** §3.1–3.2
- **摘录要点：** 从 URDF 开手构型自动建球：掌根平均定掌心、指尖距离定半径 \(r=2l/\pi\)，球心沿掌法向外 \(r\)；坐标系 \(z\) 对齐掌法、\(x\) 对齐中指方向；**按手型半径归一化到单位球**。在球面均匀采样 \((\theta,\phi)\) 与法向，投影到掌/指内表面得 **配置不变** 的稠密对应——不同手共享同一球面参数域。
- **对 wiki 的映射：**
  - [UHAS](../../wiki/methods/uhas-unified-hand-action-space.md) — 「从 URDF 到共享球坐标」的自动管线；与 RobotFingerPrint 等同族 **球面几何抽象** 的延伸（从抓取到 **操纵动作**）。

### 3) 球面形变动作参数化（15 维）与 CIK

- **链接：** §3.3–3.4
- **摘录要点：** 动作用 **驱动平面**（固定方位角 \(\theta_{\text{plane}}\)）控制侧向 \(\Delta\theta\)，平面内 **驱动向量** 控制径向 \(\Delta r\)；默认 **5 个驱动平面 × 每平面 2 个驱动向量** → **15 维连续动作**（4 指手在无名指位 **复制一环指平面** 以对齐 5 指空间）。**级联 IK**：关节分 **侧向**（主要改 \(\theta\)）与 **包络**（主要改 \(r,\phi\)）；侧向用预计算 lookup，包络沿运动链逐级 1D IK；**CIK 可达 150 Hz**。
- **对 wiki 的映射：**
  - [UHAS](../../wiki/methods/uhas-unified-hand-action-space.md) — 策略在 **UHAS** 输出形变，**球控制器 + CIK** 转关节；观测也在规范球坐标系下归一化，与动作对称。

### 4) 仿真：四手单策略 + 零样本 + 跨指数量迁移

- **链接：** §4.1–4.2
- **摘录要点：** Isaac Lab Repose Cube，四手（Allegro、LEAP、Shadow、MANO），每 episode 连续 **10 个目标朝向**；PPO（RSL-RL）轻量 actor-critic。**Multi-Hand** 四手联合训练与 **Single-Hand** 单手可比（成功率 ~99%、平均连续重定向 ~9.5–9.9）；**Zero-shot**（训练时排除目标手）在 Allegro/LEAP 仍 ~95% / ~7.7，Shadow 较弱（85.7% / 4.4）。**跨形态**：5 指组（Shadow+MANO）训的策略迁到 4 指手成功率明显下降（Allegro 66.2%），反向亦存在 gap；MANO 预训练 **500 iter 微调** 可快速恢复新机性能（相对 scratch ~4500 iter）。
- **对 wiki 的映射：**
  - [UHAS](../../wiki/methods/uhas-unified-hand-action-space.md) — 与 [In-hand Reorientation](../../wiki/methods/in-hand-reorientation.md) 任务对齐的 **跨具身 RL 实证**。
  - [跨具身策略迁移选型指南](../../wiki/queries/cross-embodiment-transfer-strategy.md) — 灵巧手层的「统一动作空间 + 多具身联合训练」数据点。

### 5) 真机：LEAP Hand sim2real 与单策略控四手演示

- **链接：** §4.3；项目页视频
- **摘录要点：** 真机 LEAP + AprilTag 立方体位姿估计；系统辨识 + DR。10 次试验 **平均连续重定向**：关节空间基线 **0.6**，UHAS 零样本 **0.9**，四手联合训 **1.1**，仅 LEAP 训 **2.0**。项目页视频展示 **同一策略同时驱动四只不同手**（Allegro、LEAP、Shadow、MANO）。
- **对 wiki 的映射：**
  - [UHAS](../../wiki/methods/uhas-unified-hand-action-space.md) — 真机 gap 与 **多手训练更保守** 的工程阅读点。
