# 智星云（AI Galaxy）

> 来源归档

- **标题：** 智星云 GPU 云服务器
- **类型：** site（国内 GPU 云算力平台）
- **来源：** 上海亘聪信息科技有限公司（安诺其集团子公司）
- **链接：** https://ai-galaxy.cn/
- **入库日期：** 2026-07-02
- **一句话说明：** 2019 年成立的 GPU 算力服务平台，提供云主机、云容器（NVLink）、裸金属与 AI 工作站；强调物理 GPU 独享、推理/训练算力池分区与高校客户占比。

## 为什么值得保留

- **产品线分层**：除常规定价云主机外，有 **裸金属**（自动驾驶仿真等重负载）与 **NVLink 云容器**（多卡大模型）。
- **卡型覆盖广**：3090、4090、A5000、A6000、A40、V100、A100 等分区展示。
- **企业/高校服务**：企业认证、1V1 服务与长期多台折扣；官网称 7 万+ 高校用户。

## 平台要点（官网 2026-07 摘录）

| 产品 | 定位 |
|------|------|
| **云主机** | 虚拟化弹性实例，按小时/日/月 |
| **裸金属** | 整机独占，适合自动驾驶仿真、工业级 HPC |
| **云容器** | NVLink/NVSwitch 多卡，大模型训练 |
| **AI 工作站** | 1–4 卡可定制，本地+云端协同 |

参考时租（官网展示，随活动变动）：3090 ¥1.10/h、4090 ¥1.65/h、A100 80G ¥6.6/h。
