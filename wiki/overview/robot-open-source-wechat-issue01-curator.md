# 机器人开源宝库（微信策展第01期）— 索引

## 一句话定义

本页把第三方微信清单中的 **10 个开源机器人/平台**拆成独立实体节点，并在此集中列出**官方文档与组织级 GitHub**入口，避免把网盘短链当作唯一分发源。

## 为什么保留这一索引

- **入口分散**：同一品牌下常有「文档站 / 组织 GitHub / 教育 eManual / 网盘物料」多条路径，初学者易漏。
- **与硬件选型页衔接**：可与 [开源人形硬件方案对比](../entities/open-source-humanoid-hardware.md)、[人形机器人](../entities/humanoid-robot.md) 对照阅读。

## 常见误区

- **误区：「微信里给的网盘 = 官方唯一源」。** 网盘路径与提取码可能变更；STEP/BOM 应以 **GitHub Release 或官网下载页** 交叉验证。
- **误区：「10 个都是人形」。** 清单含 **四足、轮式、机械臂/手**；选型时请回到任务定义（locomotion / manipulation / research）。
