# UniStore（unistore.unitree.com）

> 来源归档

- **标题：** UniStore — 宇树应用平台 / 人形机器人任务动作应用商店
- **类型：** site（官方应用生态门户）
- **URL：** <https://unistore.unitree.com/>
- **机构：** 宇树科技（Unitree Robotics）
- **入库日期：** 2026-07-08
- **一句话说明：** 宇树面向 G1 / H1 / B2 / Go2 等机型的 **云端动作与应用分发平台**：用户通过手机 App 像装手机 App 一样 **一键下载并部署** 舞蹈、武术、任务技能等运动算法包；含 **用户广场、动作库、数据集、开发者中心** 四大模块，2026-05-07 起全面向公众与开发者开放。

## 归档说明（抓取限制）

2026-07-08 自本仓库维护环境访问 `unistore.unitree.com` 及其子路径（含 `/datasets`、`/manuals`）时，站点由 **腾讯云 EdgeOne** 返回 **567 安全策略拦截页**（非 HTTP 404）。以下要点来自 **宇树官方公开报道** 与 **第三方科技媒体对开放日的交叉核对**；后续若可直连官网，应补抓左侧手册与动作详情页。

| 时间 | 里程碑 |

## 四大核心模块（官方口径）

| 模块 | 功能要点 |
|------|----------|
| **用户广场** | 用户分享、社区互动与技能展示入口 |
| **动作库** | 官方与用户上传的 **运动算法 / 任务动作包** 浏览、下载、安装 |
| **数据集** | 聚合真机采集数据；支持用户上传或下载他人共享数据集用于训练与优化 |
| **开发者中心** | SDK、原创动作封装上架、审核与 **收益分成** |

## 技术架构与部署链路（公开报道归纳）

- **后端：** 分布式 **微服务** 架构（36Kr / 车智，2025-12-15）。
- **终端：** 用户通过 **Unitree Explore** 等手机 App 连接机器人，从云端 **一键下发** 控制算法到机载终端，类比手机应用商店安装。
- **零代码：** 面向终端用户强调无需底层编程即可部署舞蹈、武术、表演等技能包。

## G1 使用门槛（2026-05 开放日公告）

| 项目 | 要求 |
|------|------|
| 手机 App | **Unitree Explore**（G1）版本 **≥ 1.9.0** |
| 机器人固件 | G1 **OTA ≥ 1.4.8**（在 App 连接机器人界面查看） |
| 审核 | 动作/版本审核约 **1–5 个工作日**；结果经消息中心通知 |

## 适配机型

- **G1**、**H1**、**B2**、**Go2** 等宇树主力机型（官方多次口径一致）。
- 公测代表应用 **「李小龙」**：在 G1 上还原截拳道动作库，可在 **走跑模式** 与 **武术状态** 间一键切换（2025-12 首发报道）。

## 2026-05 全面开放时动作库样例（24 个限时免费）

杰克逊舞、整活扭动机器人、欢呼、蹦迪、比心、螳螂拳、查尔斯顿舞等（智东西 2026-05-07 报道枚举）。
