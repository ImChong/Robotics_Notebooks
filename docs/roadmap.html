路线与论文怎么用

初学者

先按上方「阶段」搭好控制概念骨架，再在本区每次只跟一条子线读 2–4 篇代表作，读完后回到 知识图谱 把概念挂回全站。

有经验的运动控制工程师

可把本页当速查：阶段区对齐培训与评审话术，论文区按 WBC/MPC、RL、Sim2Real、IL 快速跳到站内整理好的论文列表（detail）。

1. Motion Control：运动控制主线

控制主线越稳，后面看学习方法（RL/IL）越不容易飘。

WBC / QP / TSID

重点看任务优先级、接触力分配与全身协调。这是人形控制论文的基本语法。

详细列表 →

Model Predictive Control

看如何将动力学约束写成优化问题并在线实时求解。代表作：MIT Cheetah 系列、OCS2 工作。

MPC 列表 →

Optimal Control Theory

控制理论的数学基石。包括 DP、极大值原理与 DDP/iLQR 的推导。

理论列表 →

2. RL：机器人强化学习主线

适合想真正理解 Humanoid Locomotion、鲁棒训练与策略部署的人。

Locomotion 代表作

PPO 基线、RMA 快速自适应、AMP 对抗运动先验。不仅仅是「走」，更要看如何「稳」。

详细列表 →

Latent Skill Prior

ASE, CALM, MimicKit。当你关心「既稳又自然」的行为时，这些技能嵌入工作是核心。

详细列表 →

3. Sim2Real：从仿真到真机

最能体现工程含金量的部分，弥合仿真和现实的物理失配。

Domain Randomization

物理参数随机化 (DR)、视觉域随机化。重点看「随机了什么」以及「为什么有效」。

详细列表 →

System Identification

SAGE 执行器建模、延迟建模与参数辨识。工程部署能不能站住，往往就卡在这里。

详细列表 →

4. IL & Humanoid：模仿学习与人形专项

连接动捕数据、遥操作与全身技能协调，是后续具身智能的高价值方向。

Imitation Learning

BC / DAgger 基础、Diffusion Policy、动作迁移 (Retargeting) 完整 Pipeline。

详细列表 →

Hardware & Survey

人形硬件平台演进、机器人学习领域综述。看清行业全景，避免陷入局部最优。

详细列表 →
