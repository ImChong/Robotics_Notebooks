# 机器人顶会顶刊 — 官方一手资料索引

- **类型**：site（会议 / 期刊官方介绍与投稿入口）
- **归档日期**：2026-05-22
- **覆盖实体**：ICRA、IROS、CoRL、RSS、IEEE T-RO、IJRR、Science Robotics

## 一句话说明

汇总上述 **7 个** 机器人领域常用发表渠道的 **主办方官方页面**、**当届会议站**、**论文集 / 投稿系统** 与 **作者指南**，作为选题、投稿与引用时的 **一手溯源入口**（不替代最终 PDF 或 DBLP 条目）。

## 为什么值得保留

- 研究者常混用简称（ICRA / IROS / RSS / CoRL）与期刊名（T-RO / IJRR / Sci. Robot.），官方站点的 **定位陈述、赞助方、审稿形态、论文集托管** 各不相同。
- 会议 **届次站**（如 `2026.ieee-icra.org`）每年更换，而 **学会常设页**（IEEE RAS）长期有效；本索引同时保留两类链接。
- 后续 ingest 单篇论文时，可回链本页统一维护「发表渠道」元数据。

## 1. ICRA — IEEE International Conference on Robotics and Automation

| 项目 | 内容 |
|------|------|
| **全称** | IEEE International Conference on Robotics and Automation |
| **主办方** | [IEEE Robotics and Automation Society (RAS)](https://www.ieee-ras.org/) **全额赞助**（fully sponsored） |
| **定位（官方）** | IEEE RAS **旗舰会议**（flagship conference）；汇聚全球机器人与自动化研究者讨论最新创新与突破 |
| **频率 / 形式** | 年度；主会场 + 专题分会、workshop / tutorial、展览、竞赛、社交活动等 |
| **论文集** | 录用论文发表于 **IEEE Xplore** 会议论文集（经 ICRA 审稿流程） |

### 一手网址

| 用途 | URL |
|------|-----|
| IEEE RAS 常设介绍（含重要日期、当届站链接） | https://www.ieee-ras.org/conferences-workshops/fully-sponsored/icra/ |
| 当届会议站（示例：ICRA 2026，维也纳） | https://2026.ieee-icra.org/ |
| 投稿系统（RAS PaperPlaza，与 IROS 等共用生态） | https://ras.papercept.net/ |

### 官方摘录要点（IEEE RAS ICRA 页，2026 届）

- ICRA 2026：**2026 年 6 月 1–5 日**，奥地利维也纳。
- 重要日期示例（太平洋时间 23:59）：论文开放投稿 2025-07-15、全文截止 2025-09-15、录用通知 2026-01-31、终稿 2026-03-06（以当届站为准）。

## 5. T-RO — IEEE Transactions on Robotics

| 项目 | 内容 |
|------|------|
| **全称** | IEEE Transactions on Robotics |
| **主办方** | **IEEE RAS** 赞助期刊 |
| **定位（官方）** | 发表代表机器人 **各领域重大进展** 的研究论文；欢迎理论、设计、实验、分析、算法及集成应用案例的组合 |
| **形态** | **期刊**（长文、修订轮次），非会议论文集 |
| **发表** | **IEEE Xplore**；可选开放获取（hybrid OA，见 Information for Authors） |

### 一手网址

| 用途 | URL |
|------|-----|
| RAS 期刊主页 | https://www.ieee-ras.org/publications/t-ro/ |
| Information for Authors | https://www.ieee-ras.org/publications/t-ro/t-ro-information-for-authors/ |
| 投稿系统（PaperCept） | http://ras.papercept.net/journals/tro |
| IEEE Author Center（模板与终稿） | https://ieeeauthorcenter.ieee.org/ |
| IEEE Xplore 期刊页 | https://ieeexplore.ieee.org/xpl/RecentIssue.jsp?punumber=8860 |

### 官方摘录要点

- 符合条件的 T-RO 论文可在 **ICRA、IROS、CASE** 等 RAS 相关会议上做 **期刊论文报告**（Journal Paper Presentation），细则见 T-RO 主页。
- 录用后须在编辑决定后 **30 日内** 提交终稿至 PaperCept。

## 6. IJRR — The International Journal of Robotics Research

| 项目 | 内容 |
|------|------|
| **全称** | The International Journal of Robotics Research |
| **主办方** | **SAGE Publications** |
| **定位（官方）** | **首个** 机器人研究学术期刊（1982 年创刊）；发表 **具长期档案价值** 的原创研究与综述 |
| **形态** | 期刊；含 Original Articles、Review Articles、Special Issues、Data Papers 等（投稿指南） |
| **发表** | SAGE Journals 平台；提供 **Multimedia Archives** |

### 一手网址

| 用途 | URL |
|------|-----|
| 期刊主页 | https://journals.sagepub.com/home/ijr |
| 各期目录 | https://journals.sagepub.com/loi/ijr |
| Author Instructions / Submission Guidelines | https://journals.sagepub.com/author-instructions/ijr |
| 在线投稿（ScholarOne） | https://mc.manuscriptcentral.com/ijrr |

- 强调 **原创、扎实、可供他人继续构建** 的工作。

## 7. Science Robotics

| 项目 | 内容 |
|------|------|
| **全称** | Science Robotics |
| **主办方** | **American Association for the Advancement of Science (AAAS)** |
| **定位（官方）** | **多学科** 研究期刊，覆盖传统机器人学科及与机器人密切相关的新兴技术；口号 **「Science for robotics and robotics for science」** |
| **创刊** | **2016** 年（创刊社论等见 science.org） |
| **形态** | 期刊；含 Research Articles、Reviews；亦涵盖政策、伦理与社会议题（官方创刊愿景） |

### 一手网址

| 用途 | URL |
|------|-----|
| 期刊主页（Mission & Scope） | https://www.science.org/journal/scirobotics |
| About / Information for Authors | https://www.science.org/content/page/science-robotics-journal-information-authors |
| 创刊社论（愿景） | https://www.science.org/doi/10.1126/scirobotics.aal2099 |
| Grand Challenges 专文 | https://www.science.org/doi/10.1126/scirobotics.aar7650 |

### 官方摘录要点

- 强调 **机器人设计、理论与应用** 的重要进展；尺度从纳米到宏观，涵盖医疗、工业、陆海空天与服务机器人等。
- 与侧重工程的 RAS 期刊形成 **互补**，并吸纳材料、神经科学、生物启发等交叉方向（五周年社论表述）。
