# 辉羲智能（rhino.auto）

> 来源归档

- **标题：** 辉羲智能官网
- **类型：** site
- **官方入口：** <https://rhino.auto/>（`www.rhino.auto` 同域；部分抓取环境需带 `www`）
- **入库日期：** 2026-07-17
- **一句话说明：** 北京亦庄车载 **高阶智能驾驶芯片** 与 **全栈智驾方案** 公司：2024 WICV 发布 **光至 R1**（7nm、>500 TOPS NPU、ASIL-D），并推出无图城区方案 **RINA**；公开叙事将 **具身智能机器人** 列为除智驾外的算力应用场景。

## 策展要点（公开资料汇总，非官网全文转存）

- **光至 R1（2024-10 发布）：** 7nm 车规工艺、约 **450 亿晶体管**、**8 核 SIMT NPU**（>500 TOPS DL 算力）、**24× Cortex-A78AE**（>420k DMIPS）；宣称 **原生适配 Transformer** 大模型；通过 **ASIL-D / EVITA Full** 等车规认证。
- **RINA 参考方案：** 城区 **无图** 端到端感知规划；公开材料称约 **60% 算力裕量**、系统成本降 **40%**，目标 **30 天算法迁移 / 12 个月量产交付**。
- **具身智能延伸：** 新华网等公开报道明确将 **家庭服务机器人、智能工业设备** 等列为 R1 算力场景——与纯车规 SoC 选型页不同，本库从 **边缘大算力 + 安全认证** 角度索引，供移动机器人/无人车感知栈对照。
- **产业背景：** 亦庄产投、蔚来、小米、经纬恒润等投资/合作方出现在公开报道；**无机器人整机或开源 SDK 仓库**（截至入库日官网未见 GitHub 入口）。
