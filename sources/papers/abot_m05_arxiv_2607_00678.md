- **缩写：** **ABot-M0.5**

- **arXiv：** <https://arxiv.org/abs/2607.00678>（HTML：<https://arxiv.org/html/2607.00678v1>）

- **代码：** <https://github.com/amap-cvlab/ABot-Manipulation>（M0.5 权重与代码 **coming soon**；仓库含 ABot-M0 训练/推理与部署）

- **一句话说明：** 面向 **移动操作（mobile manipulation）** 的 **World Action Model**：以 **帧级 latent action** 桥接粗粒度视频 latent 与细粒度控制，用 **双层 Mixture-of-Transformers（D-MoT）** 解耦 **移动 vs 操作** 动作子空间，并以 **Dream Forcing** 在 **自生成视频** 上训练逆动力学以对齐自回归推理；在 **RoboCasa365 / RoboTwin 2.0 / LIBERO / LIBERO-Plus** 与真机长程任务上报告 SOTA 或领先表现。

## 核心论文摘录（MVP）

### 1) 问题与三层对齐原则（Abstract / §1–2）

- **链接：** <https://arxiv.org/abs/2607.00678>
- **核心贡献：** 移动操作要求 **时间粒度、动作空间、训练–测试条件** 三层对齐，而现有 **反应式 VLA** 缺显式世界建模，既有 **WAM** 又常 **粗视频 chunk + 纠缠导航–操作动作 + GT 未来监督逆动力学**，导致 **接触细节丢失、动作分布冲突、长程 rollout 误差累积**。ABot-M0.5 提出 **Video → Latent Action → Action** 三级级联，并在 **时间 / 动作结构 / rollout 条件** 上分别用 **中间 latent action、双层 D-MoT、Dream Forcing** 对齐。

### 2) 架构：三级级联 + D-MoT + CFM（§3）

- **链接：** <https://arxiv.org/html/2607.00678v1>
- **核心贡献：** 基于 **Wan2.2** 视频扩散骨干；**3D VAE** 压缩多视角观测为 **video latent** $z_{t+1}$，**冻结 ALAM 预训练 encoder** 从相邻帧提取 **帧级 latent action** $m_t$（具身无关的局部视觉转移），再解码为 **可执行动作** $a_t$。三流 token $[X^z, X^m, X^a]$ 经 **结构化稀疏注意力**（video 不可 attend latent action；action 可 attend latent action）与 **D-MoT**：**模态级** 独立投影/头 + **动作级** 将 $a_t$ 拆为 **mobility** $a^{\mathrm{move}}$ 与 **manipulation** $a^{\mathrm{manip}}$ 子塔；全阶段用 **Conditional Flow Matching（CFM）** 统一生成目标。
- **对 wiki 的映射：**
  - [Generative World Models](../../wiki/methods/generative-world-models.md)
  - [VLA](../../wiki/methods/vla.md)（对照：反应式 vs 显式世界–动作联合）

### 3) Dream Forcing：训练–推理对齐（§3.4 / §4.4）

- **核心贡献：** 相对 **Teacher Forcing**（动作在 GT 未来视频上训练）与 **Diffusion Forcing**（训练时噪声组合难复现推理轨迹），**Dream Forcing** 分两阶段前向：**Phase A** 用少步去噪生成 **自 dreamed** 的 $\hat{z}_{t+1}, \hat{m}_t$；**Phase B** 仅在这些 dreamed latent 上优化动作 CFM——使逆动力学在 **与部署一致的不完美未来** 上学习。渐进 SFT：**SFT1** 在 GT 未来上联合训 $z,m,a$；**SFT2** 切换 Dream Forcing。RoboCasa365 Target 100% 消融：共享 50k warm-start 后，**+5k Dream Forcing → 70.56%** vs 继续 Teacher Forcing **5k → 66.78%**。

### 4) 三阶段训练与数据（§4）

- **世界模型预训练：** 自 **Wan2.2 5B** 初始化，在 **OXE / OXE-AugE / Agibot-Beta / RoboCOIN / RoboMind / Galaxea / InternData-A1** 等异构机器人数据上做 **无动作条件** 自回归未来视频预测；**四槽位固定语义相机分配**（2×第三人称 + 2×腕部）+ 缺视角零填充与 attention mask。
- **Latent action 预训练：** 采用 **ALAM** 框架，加 **加性/可逆** 转移一致性约束，冻结 $E_m$ 离线标注轨迹。
- **渐进 SFT：** SFT1 联合 $z,m,a$（teacher-forced 未来）；SFT2 Dream Forcing。工程侧：**结构化 FlashAttention**（约 **5×** 加速）与 **offset-based latent augmentation**。

### 5) 实验要点（§5）

- **RoboCasa365（pretrain 设定）：** 平均 **40.4%**（Atomic-Seen **75.9%**，Composite-Seen **38.3%**）；+Condensed Memory **46.6%**；显著高于 π₀.₅（16.9%）、GR00T-N1.5（23.9%）、GigaWorld-Policy（20.7%）、RLDX-1（33.2%）、Qwen-RobotManip（35.9%）。
- **RoboCasa365 Target 100%：** **54.2%** 平均 vs Lingbot-VA **45.1%**、Fast-WAM **43.5%**、GR00T-N1.5 **43.7%**；Target 10%：**30.1%** vs GR00T-N1.5 **21.0%**；Wan2.2 直接 SFT 仅 **17.8%**（预训练 gap **31.2%**）。
- **RoboTwin 2.0：** Clean **94.0%** / Randomized **94.2%**，平均 **94.10%**（略高于 G0.5、Qwen-RobotManip 等）。
- **LIBERO：** 平均 **99.4%**；**LIBERO-Plus（零样本）：** Total **83.4%**，在 WAM 对照中领先 Cosmos-Policy（82.2%）、ImageWAM（83.1%）、Being-H0.7（82.1%）。
- **真机（Agilex Piper 单臂，每任务 50 demo）：** 精细 **Peg Cylinder** 成功率 **70%** / 过程分 **96%**（π₀.₅ 50%/90%，Fast-WAM 30%/77%）；长程 **摆盘/摆果/摆花** 等成功率 **60–80%**，过程分 **>88%**。
- **消融（RoboTwin Clean）：** 无 latent action 基线 **87.60%** → 三阶段分离 + $p_{\mathrm{drop}}=0$ **94.00%**；动作解耦 MoT 在 Composite-Seen 长程任务上 **0.48** vs 模态级 MoT **0.34**。

### 6) 谱系与局限（§6 / Related Work）

- **谱系：** 同团队前序 **[ABot-M0](https://arxiv.org/abs/2602.11236)**（VLA foundation + action manifold）；与 **Fast-WAM、Motus、Lingbot-VA、Cosmos Policy、GigaWorld-Policy** 等同为 2026 移动/操纵 WAM 对照线；视频骨干 **Wan2.2**，latent action 来自 **ALAM**。
- **局限：** M0.5 权重/代码尚未全量开源（2026-07-01 技术报告）；真机目前以 **单臂 Agilex Piper** 为主；**Condensed Memory** 为预告扩展；长程仍依赖历史 GT 观测 grounding（closed-loop 仅 dream 最新 chunk）。
