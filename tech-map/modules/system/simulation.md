# 仿真（系统集成层）

仿真是控制、学习与部署之间的中介层：策略训练、控制器调试与场景复现多在仿真中完成，真机落地前通常还要经过 [Sim2Real](../../../wiki/concepts/sim2real.md)、[System Identification](../../../wiki/concepts/system-identification.md) 等桥接。
