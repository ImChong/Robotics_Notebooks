# FLAP: FOV-Constrained Active Perception Planning for Prior-Map-Free 3D Navigation

> 来源归档

- **标题：** FLAP: FOV-Constrained Active Perception Planning for Prior-Map-Free 3D Navigation
- **类型：** paper
- **出处：** 2026 · arXiv preprint
- **论文链接：** <https://arxiv.org/abs/2606.17630>
- **机构：** 浙江大学工业控制技术国家重点实验室 / 湖州研究院
- **入库日期：** 2026-06-28
- **一句话说明：** ZJU 提出的 **无先验地图 3D UAV 轨迹优化** 框架：在 **传感器坐标系** 从动力学推导 **FOV 可见性约束**，用 **速度触发** 的风险感知主动感知惩罚与 **可优化 AP 子段** 平衡安全与效率；双 ESDF 划分已知/未知空间，MINCO 联合时空优化；仿真与真机（Livox Mid-360 / 深度相机）验证窄垂直 FOV 与复杂竖向机动场景。

## 核心摘录（策展，非全文）

### 问题与动机

- **无先验地图**  cluttered 3D 环境中，机载传感器 **FOV 与量程有限**，未知区域碰撞风险与运动效率难以兼顾。
- 常见做法缺陷：
  1. **环境假设**（乐观/悲观）—— 乐观易撞障，悲观易死锁或过度保守；
  2. **风险响应**（限速、双轨迹 SUPER/FASTER）—— 多不直接建模 **主动观测未知区**，垂直 FOV 盲区在竖向机动时尤为致命；
  3. **信息获取**（NBV、FOV 硬约束）—— 多限于 2D 或仅 yaw 水平感知，忽略 **全姿态动力学 + 垂直 FOV** 与 **加速度致 pitch 偏转** 的耦合。
- 现有方法常把感知当作 **末端安全检查**，而非轨迹优化中的 **可微、可主动调节** 组件；且依赖昂贵 **感知感知前端** 生成可行路径。

### 方法要点

| 维度 | FLAP |
|------|------|
| **地图表示** | 射线占据栅格 → **双 ESDF**：$E_s$（未知+障碍皆不安全）、$E_u$（仅障碍不安全）；定义 $\mathcal{D}_s$、$\mathcal{D}_u$、条件可通行 $\mathcal{D}_a=\mathcal{D}_u\setminus\mathcal{D}_s$ |
| **前端** | 加权 A*；在已知安全区优先爬升至目标高度，减少进入 $\mathcal{D}_a$ 后的重复竖向主动感知 |
| **感知约束** | 在 **传感器系** 表达可见性：水平/垂直 FOV、量程、边界法向、最小观测角；由 UAV 动力学与安装外参推导 |
| **主动感知惩罚** | **速度触发** 激活函数 $\mathcal{L}_a(\mathcal{C}_{sd})$：制动距离满足时抑制感知项，风险升高时逐步激活；避免全程限速 |
| **AP 子段** | 在已知段 $\boldsymbol{S}_s$ 内插入时长 $T_p$ 的 **主动感知段** $\boldsymbol{S}_p$，**优化起始时刻** $\rho$ 鼓励提前观测 $\boldsymbol{p}_v$ |
| **轨迹表示** | **MINCO** 微分平坦输出 $[\boldsymbol{p},\psi]$，$h=3$；边界点 $\boldsymbol{p}_u$ 在已知–未知平面 $\mathcal{B}$ 上重参数化 |
| **重规划** | 监测边界邻域 ESDF 变化、AP 段末观测失败、规划延迟 $T_R$ 下的轨迹切换 |

### 与 SOTA 对照（论文摘要）

| 方法 | 主动感知 | 垂直 FOV / 3D 机动 | 备注 |
|------|----------|-------------------|------|
| SUPER / FASTER 双轨迹 | 间接（备份轨迹） | 竖向盲区内备份轨迹易失效 | 水平窄道 30° FOV 时 SUPER 完成率 <80% |
| FM（FASTER on MINCO） | 倾斜角约束 | 极窄 FOV（0.2°）不可行 | 靠限速保安全，任务时间更长 |
| RAPTOR / NBV | 硬 FOV 或视点选择 | 多偏水平或固定模式 | 深度相机窄道对比基线 |
| **FLAP** | **嵌入轨迹优化** | **传感器系 FOV + 竖向机动** | 0.2° 垂直 FOV 仍 6/10 成功 |

### 实验摘要

- **仿真（LiDAR）**：水平窄道随垂直 FOV 收紧（90°→0.2°），FLAP 在 SUPER 失败、FM 不可行时仍部分成功；U 型迷宫、头顶障碍等需 **螺旋爬升/下降** 的 3D 场景优于 SUPER。
- **仿真（深度相机）**：78° 水平 FOV、限速 3 m/s；相对 RAPTOR/FM/NBV 在窄道与重叠竖向障碍 passage 中报告更短航时与更合理观测行为。
- **真机**：Livox Mid-360（水平/倾斜安装）、Orbbec 类深度相机；高墙翻越+谨慎下降、L 形障碍 **竖向受限 vs 不限** 对比、室外非结构化 **地面→高架** 导航。
