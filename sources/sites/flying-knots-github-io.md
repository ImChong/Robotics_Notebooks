## 与论文一致的公开主张（便于 wiki 溯源）

1. **核心叙事**：可变形体 **动态操作** 不必依赖海量示教或仿真；**单次人类演示 + 简化模型 + 任务级 ILC** 可在硬件上快速收敛。
2. **示范任务**：**Flying knot** — 非平面绳索动态打结。
3. **方法要点**：每轮用 **QP 逆模型** 将 **critical-point 任务误差** 传播为 **Bézier 命令更新**。
4. **实验主张**：**7 种绳索**、**≤10 trials 100% 成功**、绳型间 **2–5 trials 迁移**。
